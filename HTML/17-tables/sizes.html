HTML Table Sizes

HTML хүснэгтүүд нь багана, мөр, хүснэгт бүр өөр өөр хэмжээтэй байж болно.

Хүснэгтийн мөр, баганын хэмжээг тодорхойлохын тулд style атрибутын width, height элементүүдийг ашиглана.

HTML Table Width

Хүснэгтийн өргөнийг тохируулахын тулд table элементэд style шинж чанарыг нэмнэ.

Хүснэгтийн өргөнийг 100% болгож тохируулна: <table style="width:100%"></table>

Tag Description
table Хүснэгтийг тодорхойлно.
th Хүснэгтийн толгой хэсгийн нүдийг тодорхойлно.
tr Хүснэгтийн мөрийг тодорхойлно.
td Хүснэгтийн нүдийг тодорхойлно.

HTML Table Column Width

Баганын хэмжээг тохируулахын тулд th, td элементэд style шинж чанарыг нэмнэ.

Эхний баганын өргөнийг 40% болгож тохируулав: <th style="width:40%">Tag</th>

Tag Description
caption Хүснэгтийн гарчигийг тодорхойлно.
colgroup Хүснэгтийн нэг буюу хэд хэдэн баганаас бүрдэх бүлгийг форматлахыг зааж өгнө.
col colgroup элементийн багана бүрийн шинж чанарыг тохируулна.

HTML Table Row Height

Мөрийн өндрийг тохируулахын тулд хүснэгтийн мөрийн элементэд style шинж чанарыг нэмнэ.

Эгнээ бүрийн өндрийг 50 пиксел болгож тохируулав: <tr style="height:50px">Tag</tr>

Tag Description
thead Хүснэгт дэх гарчгийн агуулгыг бүлэглэнэ.
tbody Хүснэгт дэх үндсэн агуулгыг бүлэглэнэ.
tfoot Хүснэгт дэх хөл хэсгийн агуулгыг бүлэглэнэ.
