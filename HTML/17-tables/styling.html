HTML Table Styling

Хүснэгтийг илүү загварлаг харагдуулахын тулд CSS ашиглана.

HTML Table - Zebra Stripes

Хэрэв хүснэгтийн хоёр дахь мөр бүрт дэвсгэр өнгө нэмбэл тахь шиг судалтай болно.
Хүснэгтийн бусад мөрийг загварчлахын тулд :nth-child(even) сонголтыг ашиглана.

<style>
table {
border-collapse: collapse;
width: 100%;
}
th, td {
text-align: left;
padding: 8px;
}
tr:nth-child(even) {
background-color: #D6EEEE;
}
</style>

Хэрэв (even) оронд (odd) байвал 2,4,6 мөрийн оронд 1,3,5, гэх мэт мөрөнд солигдоно.

HTML Table - Vertical Zebra Stripes

Босоо тахь зураастай болгохын тулд багана болгоныг загварчилна.
Хүснэгтийн өгөгдлийн элементийг :nth-child(even) байдлаар тохируулна.

<style>
table, th, td {
border: 1px solid black;
border-collapse: collapse;
}
th:nth-child(even),td:nth-child(even)
{
background-color: #D6EEEE; }
</style>

Combine Vertical and Horizontal Zebra Stripes

Дээрх хоёр жишээн дээрх загварыг нэгтгэж мөр болон багана бүрт зураастай давхардсан эффект авч болно.

Үүний тулд nth-child() элементэд even (odd) сонгогчийг ашиглан мөр бүрдээ background-color нэмнэ.

<style>
table, th, td {
border: 1px solid black;
border-collapse: collapse; }
tr:nth-child(even) {
background-color: rgba(150, 212, 212, 0.4); }
th:nth-child(even), td:nth-child(even) {
background-color: rgba(150, 212, 212, 0.4); }
</style>

Хэрэв толгой, хүснэгтийн нүдэнд эффектийг ашиглахыг хүсвэл :nth-child() сонгогчийг th, td элементүүдийн аль алинд нь байрлуулна.

Horizontal Dividers

Хэрэв зөвхөн хүснэгтийн мөр бүрийн доод хэсэгт хүрээ хийвэл хэвтээ байдлаар тусгаарлагдсан хүснэгттэй болно.

Хэвтээ тусгаарлагчийг авахын тулд tr элементүүдэд border-bottom шинж чанарыг нэмнэ.

<style>
table {
border-collapse: collapse;
width: 100%; }
tr {
border-bottom: 1px solid #ddd; }
</style>

Hoverable Table

<tr> дээрх :hover сонгогчийг ашиглан хулгана хүрэх үед хүснэгтийн мөрүүд тодорно.
