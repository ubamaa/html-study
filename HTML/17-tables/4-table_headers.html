HTML Table Headers

HTML хүснэгтүүд нь багана, мөр бүр эсвэл олон багана/мөр толгойтой байж болно.

HTML Table Headers

Хүснэгтийн толгойг th элементийг ашиглан тодорхойлно. th элемент бүр нь хүснэгтийн нүдийг төлөөлдөг.

Tag Description
table Хүснэгтийг тодорхойлно.
th Хүснэгтийн толгой хэсгийн нүдийг тодорхойлно.

Vertical Table Headers

Эхний баганыг хүснэгтийн толгой болгон ашиглахын тулд мөр бүрийн эхний нүдийг th элементээр тодорхойлно.

Tag caption colgroup col
Description Хүснэгтийн гарчигийг тодорхойлно. Хүснэгтийн нэг буюу хэд хэдэн баганаас бүрдэх бүлгийг форматлахыг зааж өгнө. colgroup элементийн багана бүрийн шинж чанарыг тохируулна.

Align Table Headers

Хүснэгтийн анхны өгөгдлөөр толгойг тод, голд нь байрлуулдаг.

Хүснэгтийн толгойг зүүн тийш зэрэгцүүлэхийн тулд text-align CSS шинж чанарыг ашиглана.

style>
table, th, td { border: 1px solid black;
border-collapse: collapse; }
th { text-align: left; }
/style>

Header for Multiple Columns

Xоёр ба түүнээс дээш баганыг хамарсан толгойтой байж болно.
Үүний тулд th элементийн colspan шинж чанарыг ашиглана.

Tag
colspan Нүдийг олон баганатай болгоход ашиглана.
rowspan Нүдийг олон эгнээ болгоход ашиглана.

Table Caption

Хүснэгтэд гарчиг нэмж болно. Үүний тулд caption тагийг ашиглана.
