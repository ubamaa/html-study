HTML Table Borders

HTML хүснэгтүүд нь өөр өөр загвар, хэлбэрийн хүрээтэй байж болно.

How To Add a Border

Хүрээ нэмэхийн тулд table, th, td элементүүд дээр CSS border шинж чанарыг ашиглана.

(<style> table, th, td { border: 1px solid black; }</style>)

Энэ жишээн дээрх нь давхар хилийг харуулна.

Collapsed Table Borders

Энэ нь хүрээг дан хил буюу зураасан хүрээтэй болгоно.

Хүрээг дан хүрээтэй болгохын тулд border-collapse CSS шинж чанарыг collapse болгож тохируулна.

<style>
table, th, td {
border: 1px solid black;
border-collapse: collapse; }
</style>

Style Table Borders

Нүд бүрийн дэвсгэр өнгийг тохируулж, хүрээний өнгийг сольж өвөрмөц хүснэгт байгуулж болно.

<style>
table, th, td {
border: 1px solid white;
border-collapse: collapse;
}
th, td { background-color: #96D4D4; }
</style>

Үл үзэгдэх хүрээ мэт сэтгэгдэл төрүүлэхийн тулд цагаан хүрээ, цэнхэр дэвсгэр өнгөөр тохируулав.

Round Table Borders

border-radius шинж чанар нь хүрээнүүдэд бөөрөнхий булан нэмнэ.

<style>
table, th, td {
border: 1px solid black;
border-radius: 10px; }
</style>

Дээрх жишээний table элементийг хасвал, гадна хүрээ хасагдаж нүднүүдэд дугуйрсан булан нэмнэ.

<style>
th, td {
border: 1px solid black;
border-radius: 10px; }
</style>

Dotted Table Borders

border-style тусламжтайгаар хүрээний харагдах байдлыг тохируулж болно.

<style>
th, td {
border-style: dotted; }
</style>

border-style утгуудыг доор жагсаая:
- dotted (цэгэн хүрээ)
- dashed (тасархай зураасан хүрээ)
- solid (зураасан хүрээ)
- double (давхар зураасан хүрээ)
- groove (ховилон хүрээ)
- ridge (жаазан хүрээ)
- inset (оруулга хүрээ)
- outset (эхлэлгүй хүрээ)
- none (хүрээ байхгүй)
- hidden (нуугдсан хүрээ)

Border Color

border-color шинж чанараар хүрээний өнгийг тохируулж болно.

<style>
th, td {
border-style:solid;
border-color: #96D4D4; }
</style>
