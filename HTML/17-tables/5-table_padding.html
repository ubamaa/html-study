HTML Table Padding & Spacing

HTML хүснэгтийн нүднүүдийн доторх дүүргэлт, мөн нүдний хоорондох зайг тохируулах боломжтой.

HTML Table - Cell Padding

Cell padding гэдэг нь агуулга болон хүснэгтийн хүрээ хоорондох зай юм.
Анхны өгөгдлөөр үүнийг 0 болгож тохируулсан байдаг.

Хүснэгтийн нүднүүдэд зай нэмэхийн тулд padding CSS шинж чанарыг ашиглана.

Padding нэмэхийн тулд padding-top, padding-bottom, padding-left, padding-right шинж чанарыг ашиглана.

Table Padding and Spacing
Cell Padding Cell Spacing
padding-top border-spacing
padding-bottom
padding-left
padding-right

Доорх утгаар дээд хүснэгтийг байгуулав.

style>
table, th, td {
border: 1px solid black;
border-collapse: collapse; }
th, td {
padding-top: 5px;
padding-bottom: 5px;
padding-left: 20px;
padding-right: 20px; }
/style>

HTML Table - Cell Spacing

Cell Spacing гэдэг нь нүд бүрийн хоорондох зай юм.
Анхны өгөгдлөөр энэ зайг 2 пикселээр тохируулсан.

Хүснэгтийн нүднүүдийн хоорондын зайг өөрчлөхийн тулд table элементэд border-spacing CSS шинж чанарыг ашиглана.

style>
table, th, td {
border: 1px solid black; }
table { border-spacing: 30px; }
/style>
