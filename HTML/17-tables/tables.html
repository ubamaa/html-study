HTML Tables

HTML хүснэгтүүд нь вэб хөгжүүлэгчдэд өгөгдлийг мөр, багана болгон цэгцлэх боломжийг олгодог.

Define an HTML Table (HTML хүснэгт байгуулах)

HTML хэл дээрх хүснэгт нь мөр, баганын доторх нүднүүдээс бүрдэнэ.

td - хүснэгтийн нүд.
tr - хүснэгтийн мөр.
th - хүснэгтийн толгой.

Table Cells

Хүснэгтийн нүд бүрийг <td> болон </td> таг-уудаар тодорхойлно.
Хүснэгтийн нүднүүдийн агуулга энэ хооронд байгаа бүх зүйл.

Он сар өдөр Шинэ үгс Утга Жишээ

Table Rows

Хүснэгтийн мөр бүр нь <tr> -аар эхэлж </tr> таг-аар төгсдөг.
tr хүснэгтийн эгнээ гэсэн үг.

Он сар өдөр Шинэ үгс Утга Жишээ
2023.03.27 semla маслотай жижиг талх köpa semlor

Table Headers

th хүснэгтийн толгой хэсгийг илэрхийлнэ.
th элементийн текст тод, голд байрладаг иймд td тагыг ашигладаг.

Он сар өдөр Шинэ үгс Утга Жишээ
