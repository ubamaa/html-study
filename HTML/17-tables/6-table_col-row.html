HTML Table Colspan & Rowspan

HTML хүснэгт нь олон мөр эсвэл багануудыг хамарсан нүдтэй байж болно.

HTML Table - Colspan

Нүдийг олон баганатай болгохын тулд colspan шинж чанарыг ашиглана.

colspan шинж чанарын утга нь хүрээлэх баганын тоо байна.

HTML Table - Rowspan

Нүдний зайг олон эгнээ болгохын тулд rowspan шинж чанарыг ашиглана.

rowspan атрибутын утга нь хүрээлэх мөрийн тоог илэрхийлнэ.
