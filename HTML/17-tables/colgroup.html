HTML Table Colgroup

Хэрэв хүснэгтийн эхний хоёр баганыг загварчлахыг хүсвэл colgroup, col элементүүдийг ашиглана.

colgroup элементийг баганыг тодорхойлохын тулд сав болгон ашиглах ёстой.

Colgroup-t col элементийг зааж өгнө. span шинж чанар нь хэдэн баганад загвар орохыг тодорхойлдог.
Style шинж чанар нь багануудын загварыг тодорхойлно.

<table style="width: 100%;">
<colgroup>
<col span="2" style="background-color: #D6EEEE">
</colgroup>

</colgroup> таг нь table элементийн дэд элемент нь бөгөөд thead, tr, td гэх мэт
бусад элементүүдийн өмнө, гэхдээ caption элементийн дараа байх ёстой.

Legal CSS Properties

Colgroup дээр зөвшөөрөгдсөн CSS шинж чанарууд маш хязгаарлагдмал сонголттой:

CSS properties
width Элементийн өргөнийг тохируулдаг.
visibility Тухайн элементийн харагдах эсэхийг заадаг.
background Дэвсгэр шинж чанаруудыг тохируулна.
border Хүрээний хэв маягийг тохируулна.

Эдгээрээс бусад CSS шинж чанарууд хүснэгтэд нөлөөлөхгүй.

Multiple Col Elements

Хэрэв өөр хэв маягтай баганыг загварчлахыг хүсвэл colgroup дотор илүү олон col элемент ашиглана.

<table style="width: 100%;">
<colgroup>
<col span="2" style="background-color: #D6EEEE">
<col span="3" style="background-color: pink">
</colgroup>

Энэ хүснэгт эхний 2 багана, дараагийн дараалсан 3 багануудад (3,4,5) өнгө нэмсэн.

Empty Colgroups

Хэрэв хүснэгтийн дундах багануудыг загварчлахыг хүсвэл өмнөх баганад нь col "empty" элемент оруулна.

<table style="width: 100%;">
<colgroup>
<col span="3">
<col span="2" style="background-color: pink">
</colgroup>

Энэ хүснэгт эхний 3-н багана загваргүй, 4 болон 5 багануудад өнгө оруулсан байна.

Hide Columns

visibility: collapse шинж чанаруудын тусламжтай баганыг нууж болно.

<table style="width: 100%;">
<colgroup>
<col span="2">
col span="3" style="visibility: collapse">
</colgroup>

Энэ хүснэгтийн 3 баганыг (3,4,5) нуусан харагдаж байна.
