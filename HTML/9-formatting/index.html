HTML Text Formatting

HTML нь тусгай утгатай текстийг тодорхойлох хэд хэдэн элементүүдийг агуулдаг.

HTML Formatting Elements

b - Тод текст

strong - Чухал текст

i - Налуу текст

em - Онцолсон текст

mark - Тэмдэглэгдсэн текст

small - Жижиг текст

del - Устгасан текст

ins - Оруулсан текст

sub - Дэд бичвэр

sup - Дээд талын бичвэр

HTML b and strong Elements

HTML <b> элемент нь текстийг тодотгож өгдөг.

Энэ тод текст - b element ашигласан.

HTML <strong> элемент нь чухал ач холбогдолтой текстийг тодорхойлдог.

Энэ чухал, ач холбогдолтой текст! - strong element ашигласан.

HTML i and em Elements

HTML <i> элемент нь агуулгыг налуу хэлбэрээр харуулдаг.

Энэ i элементээр техникийн нэр томъёо, өөр хэл дээрх хэллэг, бодол санааг бичихдээ ихэвчлэн ашигладаг.

HTML <em> элемент нь онцолсон текстийг тодорхойлдог.

Агуулгыг налуу хэлбэрээр харуулдаг.

HTML small Element

HTML <small> элемент нь жижиг текстийг тодорхойлдог.

HTML mark Элемент

HTML <mark> элемент нь тэмдэглэх эсвэл тодруулах шаардлагатай текстийг тодорхойлдог.

HTML del Element

HTML <del> элемент нь баримтаас устгасан текстийг тодорхойлдог.

Текст дэх устгасан үг эсвэл мөрийг дундуур зураасаар зурж тэмдэглэнэ.

HTML ins Element

HTML ins элемент нь текстэнд нэмсэн үгийг тодорхойлдог.

Текстийн устгасан үгийг залруулж оруулсан үгийн доогуур зураасаар тэмдэглэнэ.

HTML sub Element

HTML sub элемент нь дэд тэмдэгтийн текстийг тодорхойлдог.

Дэд бичвэр нь үгийн доод хэсэгт жижиг үсгийн фонтоор бичигдэнэ. Химийн томьёонд ашиглаж болно.

HTML sup Element

HTML sup элемент нь дээд бичгийн текстийг тодорхойлдог.

Энэ нь ердийн мөрөөс дээш гарч ирдэг бөгөөд жижиг фонтоор дүрслэгддэг.

Дээд талын бичвэр нь математикийн томьёонуудад өргөн хэрэглэгддэг.
