HTML id Attribute

Using the id Attribute

ID шинж чанар нь хуудасны тодорхой хэв маягийг зааж өгөхөд хэрэглэдэг.
Хэш тэмдэг(#) id нэр, {} дотор шинж чанаруудыг тодорхойлон id attribute бичнэ.

Difference Between Class and ID

Class нэрийг олон HTML элемент ашиглаж болно, id-г зөвхөн нэг HTML элемент ашиглах ёстой.

HTML Bookmarks with ID and Links

HTML bookmarks вэб хуудасны тодорхой хэсэг рүү шилжих боломжийг олгодог.
Link Bookmarks файлд id элемент ашиглагдсан.
