CSS Colors, Fonts and Sizes

CSS color шинж чанар нь текстийн өнгийг тодорхойлдог.

CSS font-family шинж чанар нь фонтыг тодорхойлдог.

CSS font-size шинж чанар нь текстийн хэмжээг тодорхойлдог.

CSS-ийн color, font-family, fot-size шинж чанаруудыг ашиглан гарчиг, мөр өөрчлөгдсөн байна.
