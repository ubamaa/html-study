HTML Styles - CSS

CSS нь Cascading Style Sheets гэсэн үгийн товчлол юм.

"Cascading" гэдэг нь хэрэв үндсэн текстийн өнгийг "цэнхэр" болгож тохируулбал гарчиг, мөр болон бусад текстийн элементүүд ижил өнгөтэй болно гэсэн үг юм.

CSS нь маш их ажлыг гүйцэтгэж, хугацаа хэмнэдэг.

CSS-ийн тусламжтайгаар өнгө, фонт, текстийн хэмжээ, элементийн хоорондын зай, элементүүдийн байршил, дэвсгэр өнгө, зураг, төхөөрөмжийн төрөл, дэлгэцийн янз бүрийн хэмжээг хянах боломжтой.

CSS-ийг HTML баримт бичигт 3-н аргаар нэмж болно:

Inline - HTML элементүүдийн доторх загварын шинж чанарыг ашиглах;

Internal - head хэсгийн style элементийг ашиглана;

External - link элементийг ашиглан гадаад CSS файлтай холбох.

Inline CSS

Inline CSS нь нэг HTML элементэд өвөрмөц хэв маягийг хэрэглэхэд ашиглагддаг.

Inline CSS нь HTML элементийн style шинж чанарыг ашигладаг.

Энд h1, p өнгүүдийг өөрчилсэн байна.
