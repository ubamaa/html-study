External CSS

External CSS-ыг олон HTML хуудасны загварыг тодорхойлоход ашигладаг.

External CSS-ыг ашиглахын тулд HTML хуудас бүрийн <head> хэсэгт холбоосыг нэмнэ.

External style хуудсыг ямар ч текст засварлагч дээр бичиж болно.

Файл нь HTML код агуулаагүй байх ёстой бөгөөд .css өргөтгөлөөр хадгалагдах ёстой.
