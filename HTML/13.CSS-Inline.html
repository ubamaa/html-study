HTML Styles - CSS

CSS нь Cascading Style Sheets гэсэн үгийн товчлол юм.

"Cascading" гэдэг үг нь хэрэв үндсэн текстийн өнгийг "цэнхэр" болгож тохируулбал бүх гарчиг, догол мөр болон бусад текстийн элементүүд ижил өнгөтэй болно гэсэн үг юм.

CSS нь маш их ажлыг хэмнэдэг.

CSS-ийн тусламжтайгаар та өнгө, фонт, текстийн хэмжээ, элементийн хоорондын зай, элементүүдийн байршил, байршил, ашигласан дэвсгэр зураг эсвэл дэвсгэр өнгө, өөр өөр төхөөрөмж, дэлгэцийн хэмжээ, өөр өөр дэлгэцийг хянах боломжтой.

CSS-ийг HTML баримт бичигт гурван аргаар нэмж болно:

Inline - HTML элементүүдийн доторх style шинж чанарыг ашиглах;

Internal - head хэсгийн style элементийг ашиглана;

External - link элементийг ашиглан гадаад CSS файлтай холбох.

Inline CSS

Inline CSS нь нэг HTML элементэд өвөрмөц хэв маягийг хэрэглэхэд ашиглагддаг.

Inline CSS нь HTML элементийн style шинж чанарыг ашигладаг.

Энд h1, p өнгүүдийг өөрчилсэн байна.
