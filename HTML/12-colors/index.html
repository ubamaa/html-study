Color Values

HTML дээр өнгийг RGB, HEX, HSL, RGBA, HSLA утгууд ашиглан зааж өгч болно.

Дараах 3-н элементийн дэвсгэр өнгө нь RGB, HEX, HSL утгуудаар тохируулагдсан байна:
("Tomato" нэртэй ижил өнгөтэй.)

rgb(255, 99, 71)

#ff6347

("Tomato" өнгөтэй адил боловч 50% илүү тунгалаг)

rgba(255, 99, 71, 0.5)

HTML RGB and RGBA Colors

RGB өнгөний утга нь (red, green, blue) гэрлийн эх үүсвэрийг илэрхийлнэ.

RGBA өнгөний утга нь RGB-н альфа суваг (тунгалаг байдал) бүхий нэмэлт утга юм.

RGB Color Values

HTML дээр RGB өнгөний утгыг дараах томьёогоор тодорхойлно:

rgb (red, green, blue)

Параметр бүр (red, green, blue) өнгөний эрчмийг 0-255 хүртэлх утгаар тодорхойлно.

rgb(255, 0, 0) - улаанаар харагдана.

rgb(0, 255, 0) - ногоон өнгөөр ​​харагдана.

rgb(0, 0, 255) - хөх харагдана.

Хар өнгийг харуулахын тулд өнгөний параметрүүдийг 0 болгож тохируулна: rgb(0, 0, 0).

Цагаан өнгийг харуулахын тулд өнгөний параметрүүдийг 255 болгож тохируулна: rgb(255, 255, 255).

Shades of Gray

rgb(200, 200, 200)- Саарал өнгө нь 3-н параметрийн ижил утгыг ашиглан тодорхойлогддог.

RGBA Color Values

RGBA утга нь RGB өнгөний альфа бүхий утгын өргөтгөл бөгөөд өнгөний тунгалаг байдлыг тодорхойлдог.

RGBA өнгөний утгыг дараах байдлаар тодорхойлно:

rgba (red, green, blur, alpha)

rgba(255, 99, 71, 0.2)- Альфа параметр нь 0.0 (тунгалаг) ба 1.0 (бараан) хоорондох тоо юм.

HTML HEX Colors

#RRGGBB - RR(red), GG(green), BB(blue) 16-тын бүхэл тоо нь өнгөний бүрэлдэхүүн хэсгүүдийг тодорхойлдог.

HEX Color Values

CSS-д өнгийг дараах хэлбэрээр арван зургаатын тоогоор зааж өгч болно:

#rrggbb

rr (red), gg (green), bb (blue) - 00-ff хүртэлх 16-тын суурьтай тоoллын утгууд (0-255 аравтын бутархай).

#ff0000 - red (ff - хамгийн дээд утга), green =0, blue = 0

#00ff00 - green (ff - хамгийн дээд утга), red = 0, blue = 0

#ffa500

#3cb371

#000000 - хар өнгө (бүх утга = 00).

#ffffff - цагаан өнгө (бүх утга = ff).

Shades of Gray

#dcdcdc- Саарал өнгө нь 3-н параметрийн ижил утгыг ашиглан тодорхойлогддог.

HTML HSL and HSLA Colors

HSL нь Hue, Saturation, Lightness гэсэн үгийн товчлол юм.

HSLA өнгөний утга нь альфа суваг (тунгалаг байдал) бүхий HSL-ийн нэмэлт өнгө юм.

HSL Color Values

hsl (hue, saturation, lightness)

Hue нь өнгөт дугуй дээрх 0-360 хүртэлх градус юм.

Saturation - хувь хэмжээний утга юм. Өнгөний эрч хүчийг тодорхойлж байгаа.
0% бүрэн саарал, 50% саарал боловч өнгөтэй, 100% бүтэн өнгө саарал өнгө байхгүй.

Lightness - хувь утга юм. Өнгөний цайвар байдал буюу гэрлийн хэмжээг тодорхойлж байгаа.
0% хар (гэрэлгүй), 50% цайвар (харанхуй эсвэл цайвар биш), 100% цагаан (бүрэн тод) байдлыг илэрхийлдэг.

Shades of Gray

Саарал өнгө нь hue болон saturation 0 болгож, lightness-г 0%-100% хүртэл өөрчлөн бараан/цайвар өнгөтэй болно.

hsl(0, 0%, 60%)

HSLA Color Values

HSLA өнгө нь тунгалаг байдлыг тодорхойлдог Альфа суваг бүхий HSL өнгөний утгын өргөтгөл юм.

hsla (hue, saturation, lightness, alpha)

hsla(9, 100%, 64%, 0.2) - Альфа параметр нь 0.0 (бүрэн тод) ба 1.0 (тод биш) хоорондох тоо юм:
