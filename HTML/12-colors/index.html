HTML Colors

HTML өнгийг урьдчилан тодорхойлсон өнгөний нэрээр эсвэл RGB, HEX, HSL, RGBA, HSLA утгуудаар зааж өгдөг.

Color Names

HTML хэл дээр өнгөний нэрийг ашиглан өнгийг тодорхойлж болно.

Background Color

HTML элементүүдийн дэвсгэр өнгийг тохируулж болно.

Text color

Мөн текстийн өнгийг тохируулж болно.

Border Color

Хүрээний өнгө

Color Values

HTML дээр өнгийг RGB, HEX, HSL, RGBA, HSLA утгууд ашиглан зааж өгч болно.

Дараах 3-н элементийн дэвсгэр өнгө нь RGB, HEX, HSL утгуудаар тохируулагдсан байна:

rgb(255, 99, 71)

#ff6347

hsl(9, 100%, 64%)

Дараах 2 элементийн дэвсгэр өнгө нь RGBA болон HSLA утгуудаар тохируулагдсан:

rgba(255, 99, 71, 0.5)

hsla(9, 100%, 64%, 0.5)

HTML RGB and RGBA Colors

RGB өнгөний утга нь (red, green, blue) гэрлийн эх үүсвэрийг илэрхийлнэ.

RGBA өнгөний утга нь RGB-н альфа суваг (тунгалаг байдал) бүхий нэмэлт утга юм.

RGB Color Values

HTML дээр RGB өнгөний утгыг дараах томьёогоор тодорхойлно:

Параметр бүр (red, green, blue) өнгөний эрчмийг 0-ээс 255 хүртэлх утгаар тодорхойлно.

rgb(255, 0, 0) - улаанаар харагдана.

rgb(0, 255, 0) - ногоон өнгөөр ​​харагдана.

rgb(0,0,255) - blue харагдана.

Хар өнгийг харуулахын тулд бүх өнгөний параметрүүдийг 0 болгож тохируулна: rgb(0, 0, 0).

Цагаан өнгийг харуулахын тулд бүх өнгөний параметрүүдийг 255 болгож тохируулна: rgb(255, 255, 255).

Shades of Gray

rgb(200, 200, 200)- Саарал өнгө нь 3-н параметрийн ижил утгыг ашиглан тодорхойлогддог.

RGBA Color Values

RGBA утга нь RGB өнгөний альфа бүхий утгын өргөтгөл бөгөөд өнгөний тунгалаг байдлыг тодорхойлдог.

rgba(255, 99, 71, 0.2)- Альфа параметр нь 0.0 (тунгалаг) ба 1.0 (бараан) хоорондох тоо юм.

HTML HEX Colors

#RRGGBB - RR (улаан), GG (ногоон), BB (цэнхэр) зургаан арван бүхэл тоо нь өнгөний бүрэлдэхүүн хэсгүүдийг тодорхойлдог.

HEX Color Values

rr (red), gg (green), bb (blue)- 00-оос ff хүртэлх 16-тын суурьтай тоoллын утгууд (0-255 аравтын тоотой адил).

HTML HSL and HSLA Colors

HSL нь Hue, Saturation, Lightness гэсэн үгийн товчлол юм.

HSLA өнгөний утга нь альфа суваг (тунгалаг байдал) бүхий HSL-ийн нэмэлт өнгө юм.

HSL Color Values

Hue нь өнгөт дугуй дээрх 0-ээс 360 хүртэлх градус юм.

Saturation- хувь хэмжээний утга юм. 0% саарал өнгөний сүүдэр, 100% бүтэн өнгө гэсэн үг.

Lightness бас хувь утга юм. 0% хар, 100% цагаан.

Shades of Gray

hsl(0, 0%, 60%)- Саарал өнгө нь hue болон saturation 0 болгож, lightness-н хувийг өөрчлөн бараан/цайвар өнгөтэй болно.

HSLA Color Values

hsla(9, 100%, 64%, 0.2)- Альфа параметр нь 0.0 (бүрэн тод) ба 1.0 (тод биш) хоорондох тоо юм:
