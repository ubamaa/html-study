HTML JavaScript

JavaScript нь HTML хуудсыг илүү динамик, интерактив болгодог.

The HTML script Tag

JavaScript хэрэглээ нь зураг боловсруулах, маягтыг баталгаажуулах, динамикаар контентыг өөрчлөхөд ашиглагддаг.

JavaScript нь document.getElementById() аргыг ихэвчлэн ашигладаг.

HTML script тагийг клиент талын скриптийг (JavaScript) тодорхойлоход ашигладаг.

script элемент нь скрипт мэдэгдлүүдийг агуулсан, мөн src атрибутаар дамжуулан гадаад скрипт файл руу чиглүүлдэг.

Энэ жишээ нь "Hello JavaScript!" гэж бичсэн id="demo"-тай HTML элемент:

p id="demo">/p>

script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
/script>

A Taste of JavaScript

JavaScript-н хийж чадах зарим жишээ энд байна:

1. JavaScript нь агуулгыг өөрчлөх боломжтой;

p id="demo">JavaScript can change the content of an HTML element/p>

script>
function myFunction() {
document.getElementById("demo").innerHTML = "Hello JavaScript!";
}
/script>
button type="button" onclick="myFunction()">Товч дарах!/button>

2. JavaScript нь хэв маягийг өөрчилж болно;

p id="demo">JavaScript can change the style of an HTML element./p>

script>
function myFunction() {
document.getElementById("demo").style.fontSize = "25px";
document.getElementById("demo").style.color = "red";
document.getElementById("demo").style.backgroundColor = "yellow";
}
/script>

button type="button" onclick="myFunction()">Click Me!/button>

3. JavaScript нь шинж чанаруудыг өөрчилж болно:
Энд JavaScript нь зургийн src (эх) шинж чанарын утгыг өөрчилсөн байгаа.

The HTML noscript Tag

HTML noscript таг нь хөтчийнхөө скриптийг идэвхгүй болгосон эсвэл
скрипт дэмждэггүй хөтөчтэй хэрэглэгчдэд харуулах өөр контентыг тодорхойлдог.

p id="demo">/p>

script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
/script>

noscript>Sorry, your browser does not support JavaScript!/noscript>

JavaScript дэмждэггүй хөтөч нь noscript элемент дотор бичигдсэн текстийг харуулах болно.
