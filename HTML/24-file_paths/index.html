HTML File Paths (HTML файлын замууд)

File Path нь вэбсайтын директорт байгаа файлын байршлыг тодорхойлдог.

Файлын замыг дараах гадаад файлуудтай холбоход ашигладаг:

  • Web pages (Цахим хуудас)
  • Images (Зургууд)
  • Style sheets (Загварын хуудас)
  • JavaScript
File Path Examples
Path Description
<img src="picture.jpg"> "picture.jpg" файл нь одоогийн хуудастай нэг folder-д байгаа.
<img src="images/picture.jpg"> "picture.jpg" файл нь одоогийн хавтсанд байгаа зургуудын хавтсанд байрладаг.
<img src="/images/picture.jpg"> "picture.jpg" файл нь одоогийн вэбсайтын үндсэн хэсэгт байрлах зургийн хавтсанд байрладаг.
<img src="../picture.jpg"> "picture.jpg" файл нь одоогийн folder-оос нэг түвшний дээгүүр хавтсанд байрладаг.

Relative File Paths

Харьцангуй файлын (Relative File) зам нь одоогийн хуудастай холбоотой файлыг заадаг.

Дараах жишээнд файлын зам нь одоогийн вэбийн үндсэн хэсэгт байрлах зургийн хавтсанд байгаа файлыг зааж байна.
<img src="/images/picture.jpg" alt="Mountain">

Дараах жишээнд файлын зам нь одоогийн хавтсанд байрлах зургийн хавтсанд байгаа файлыг зааж байна.
<img src="images/picture.jpg" alt="Mountain">

Дараах жишээнд файлын зам нь одоогийн фолдероос нэг түвшний дээрх хавтсанд байрлах зургийн хавтсанд байгаа файлыг зааж байна.
<img src="../images/picture.jpg" alt="Mountain">

Best Practice

Харьцангуй файлын (Absolute File) замыг (боломжтой бол) ашиглах нь дээр.

Харьцангуй файлын замыг ашиглах үед таны вэб хуудсууд таны одоогийн үндсэн URL-тай холбогдохгүй.
Бүх холбоосууд нь таны өөрийн компьютер дээр (localhost),
түүнчлэн таны одоогийн домэйн болон ирээдүйн нийтийн домэйн дээр ажиллах болно.
