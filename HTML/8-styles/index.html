HTML Styles

HTML style attribute нь өнгө, фонт, хэмжээ гэх мэт элементэд загвар нэмэхэд ашиглагддаг.

The HTML Style Attribute

style attribute-ийг ашиглан HTML элементийн хэв маягийг тохируулж болно.

HTML style атрибут нь дараах синтакстай байна:

<tagname style="property:value;"></tagname>

Энэ утга нь CSS утга юм.

Background Color

CSS background-color attribute нь HTML элементийн дэвсгэр өнгийг тодорхойлдог.

background-color нь бүхэл хуудасны дэвсгэрийг moccasin өнгөтэй болгосон.

Мөн тухайн гарчиг болон мөрийн background-ийг өөрчилж байна.

Text Color

CSS color attribute нь HTML элементийн текстийн өнгийг тодорхойлдог.

Мөрний өнгийг өөрчилж болно.

Fonts

CSS font-family attribute нь HTML элементэд ашиглах фонтыг тодорхойлдог.

Энэ текстийн фонтыг өөрчилж байна.

Text size

CSS font-size attribute нь HTML элементийн текстийн хэмжээг тодорхойлдог.

Гарчиг болон мөрний текстийн хэмжээг өөрчилж болно.

Text Alignment

CSS text-align attribute нь HTML элементийн текстийн байрлалыг тодорхойлдог.

Гарчиг, мөр хуудасны төвд байрласан байна.
