HTML Paragraphs

HTML Paragraphs

HTML p-элемент нь догол мөрийг тодорхойлдог.

Догол мөр нь үргэлж шинэ мөрөөс эхэлдэг.

Догол мөрний өмнө болон хойно автоматаар хоосон зай (margin) нэмдэг.

HTML Display

HTML дээр бичигдсэн зүйл хэрхэн харагдахыг бид мэдэхгүй байна.

HTML дээр нэмэлт зай эсвэл нэмэлт мөр нэмэх замаар дэлгэцийг өөрчлөх боломжгүй.

Харин доор танилцах элементүүд туслах болно.

HTML Horizontal Rules

hr - Элемент нь HTML хуудасны агуулгыг салгахад ашиглагддаг.

hr таг нь хоосон таг бөгөөд энэ нь төгсгөлийн таггүй гэсэн үг юм.

HTML Line Breaks

br - элемент нь мөр таслахыг тодорхойлдог.

Шинэ догол мөр
эхлүүлэхгүйгээр мөр таслахад ашиглана.

Таг br нь хоосон таг бөгөөд
энэ нь төгсгөлийн таггүй гэсэн үг юм.

The Poem Problem

p таг өгүүлбэр үргэлжилсэн мөрөөр бичигддэг тул шүлэг бичихэд төвөг учирдаг.

Шийдэл: The HTML pre Element

pre - элемент нь урьдчилан форматласан текстийг тодорхойлдог (жишээ нь; шүлэг, найраглал).

    Энэ элемент нь мөрийг
    шүлэг шиг хэлбэрээр бичнэ.
    Элемент доторх текстийг 
    тогтмол өргөнтэй фонтоор 
    (ихэвчлэн Courier) харуулдаг.
