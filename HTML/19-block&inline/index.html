Block level elements - Блок түвшний элементүүд

P ба DIV элемент нь хоёулаа блок элементүүд. Блок-түвшний элемент нь үргэлж шинэ мөрөнд эхэлж, бүрэн өргөнийг авах боломжтой.

p элемент нь HTML баримт бичигт догол мөрийг тодорхойлдог.

div элемент нь хуваагдал эсвэл HTML документ дээр хувийг тодорхойлдог.

Inline Elements -

span нь Inline элемент юм.

Дотоод элемент нь шинэ мөрөнд эхлэхгүй. Дотоод элемент нь зөвхөн шаардлагатай хэмжээгээр өргөн хүрээнээс их хэмжээгээр авдаг.
