HTML Block and Inline Elements

HTML элемент бүр элементийн төрлөөс хамааран өгөгдмөл дэлгэцийн утгатай байдаг.
Дэлгэцийн хоёр утга байдаг:

  • Block
  • Inline

Block-level Elements (Блок түвшний элементүүд)

Блок элемент нь үргэлж шинэ мөрөөс эхэлдэг бөгөөд элементийн өмнө, хойно автоматаар зай (margin) нэмдэг.

Түгээмэл хэрэглэгддэг блок элементүүд нь <p> болон <div> юм.

p -Элемент нь HTML баримт бичгийн догол мөрийг тодорхойлдог.

div -Элемент нь HTML баримтын хэсэг эсвэл ялгаж тодорхойлдог.

Inline Elements

Inline element нь шинэ мөрөнд эхэлдэггүй, шаардлагатай хэмжээгээр өргөнийг эзэлдэг.

span нь Inline элемент юм.

The div Element

div элементийг ихэвчлэн бусад HTML элементүүдийн контейнер болгон ашигладаг.

Энэ элемент нь шаардлагатай шинж чанарууд байхгүй ч, style, class, id эдгээртэй нийтлэг.
div элементийг CSS-тэй хамт ашиглах үед агуулгын блокуудыг загварчлахад ашиглаж болно.

The Element

span элемент нь текстийн хэсэг эсвэл баримтын хэсгийг тэмдэглэхэд ашигладаг доторлогоотой сав шиг юм.

Энэ элемент нь шаардлагатай шинж чанарууд байхгүй ч, style, class, id эдгээртэй нийтлэг.
CSS-тэй хамт ашиглах үед span элементийг текстийн зарим хэсгийг загварчлахад ашиглаж болно.
