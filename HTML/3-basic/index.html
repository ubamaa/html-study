HTML Basic Examples

Энэ бүлэгт HTML-н үндсэн жишээг харуулах болно.

HTML Documents

HTML бүх document <!DOCTYPE html> илэрхийллээр (declaration) эхлэх ёстой.

HTML document <html>-ээр эхэлж, </html>-ээр төгснө.

HTML document-н харагдах хэсэг <body> болон </body> хооронд байна.

!DOCTYPE Declaration

<!DOCTYPE> declaration баримтын төрлийг илэрхийлэх бөгөөд вэб хуудсыг зөв харуулахад тусалдаг.

Хуудасны дээд талд (HTML тагаас өмнө) зөвхөн нэг удаа гарч ирэх ёстой.

<!DOCTYPE> нь жижиг том үсэг ялгахгүй.

HTML Headings

HTML гарчиг нь <h1>-<h6> хүртэлх тагаар тодорхойлогддог.

<h1> - хамгийн чухал, <h6> нь хамгийн жижиг гарчгийг тодорхойлно.

Энэ хуудасны гарчигуудыг жишээ болгон харуулав!

HTML Paragraphs

HTML догол мөрийг <p> тодорхойлно.

HTML links

HTML холбоосууд (link) нь <a> тагаар тодорхойлогдно.

Холбоосын очих газрыг (href) attribute зааж өгнө.

Attributes нь HTML элементүүдийн талаар нэмэлт мэдээлэл өгөхөд ашиглагддаг.

This is an art.
HTML Images

HTML зургууд нь (img) тагаар тодорхойлогддог.

(src) - source file (эх файл), (alt) - alternative text, (width, height) хэмжээ заасан attributes.
