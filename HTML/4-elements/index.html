HTML elements

HTML элемент нь эхлэл таг, агуулга, төгсгөлийн тагаар тодорхойлогддог.

HTML Elements

HTML элемент нь эхлэл тагаас эхлээд төгсгөлийн таг (<tagname> ... </tagname>) хүртэлх бүх зүйл юм .

Зарим HTML элементүүд ямар ч агуулга байхгүй.

Эдгээр элементүүдийг хоосон элемент гэж нэрлэдэг. Хоосон элементүүдэд төгсгөлийн тэмдэг байхгүй!

Nested HTML Elements

Бүх HTML document нь үүрлэсэн HTML элементүүдээс бүрдэнэ.

<html> нь үндсэн элемент бөгөөд HTML баримтыг бүхэлд нь тодорхойлдог.

<html> элемент дотор <body> элемент байна.

<body> элемент дотор <h1>, <p> элементүүд байна.

Never Skip the End Tag

Төгсгөлийн тагийг мартвал зарим HTML элементүүд зөв харагдаж болно.

Хэрэв төгсгөлийн тагийг мартсан бол гэнэтийн үр дүн, алдаа гарч болзошгүй!!!

Empty HTML Elements

Агуулгагүй HTML элементүүдийг хоосон элемент гэж нэрлэдэг.

<br> - таг нь мөр таслахыг тодорхойлдог
бөгөөд хаалтын таггүй хоосон элемент юм.

HTML is Not Case Sensitive

HTML стандарт нь жижиг үсгээр бичихийг шаарддаггүй ч, HTML хэл дээр жижиг үсгээр бичихийг зөвлөдөг.

XHTML жижиг үсгээр бичихийг хатуу шаарддаг баримт бичгийн төрлүүдэд орно.
