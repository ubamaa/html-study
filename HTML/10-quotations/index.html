HTML Quotation and Citation Elements

Энэ бүлэгт blockquote, q, abbr, address, cite, bdo HTML элементүүдтэй танилцах болно.

HTML blockquote for Quotations (Эх сурвалжаас иш татсан)

HTML <blockquote> элемент нь өөр эх сурвалжаас иш татсан хэсгийг тодорхойлдог.

HTML is the standard markup language for Web pages.

HTML q for Short Quotations (Богино эшлэлүүдийн)

HTML <q> таг нь богино ишлэлийг тодорхойлдог.

Энэ нь ишлэлийн эргэн тойронд хашилт оруулдаг.

HTML abbr for Abbreviations (Товчлолын)

HTML <abbr> таг нь товчлолыг тодорхойлдог. "HTML", "CSS", "Mr", "Dr", "ASAP", "ATM" г.м.

HTML товчилсон үгэн дээр хулганаа аваачихад бүтнээрээ харагдана.

HTML address for Contact Information (Холбоо барих мэдээллийн)

HTML <address> таг нь баримт бичиг эсвэл зохиогч/эзэмшигчтэй харилцах мэдээллийг тодорхойлдог.

Холбоо барих мэдээлэл нь имэйл хаяг, URL, хаяг, утасны дугаар, олон нийтийн мэдээллийн хэрэгсэл байж болно.

HTML cite for Work Title (Ажлын гарчигт зориулсан)

HTML <cite> таг нь бүтээлч ажлын (ном, шүлэг, дуу, кино, уран зураг, баримал г.м.) нэрийг тодорхойлдог.

Чемпионы Анна и Камила. 2021
