HTML Favicon

Favicon нь таб дахь хуудасны гарчгийн хажууд харагдах жижиг зураг юм.

How To Add a Favicon in HTML

Favicon нь жижиг зураг тул өндөр тодосгогчтой энгийн зураг байх ёстой.
Зургийн нийтлэг нэр нь "favicon.ico" байх юм.

index.html файлдаа title элемент доор link элемент нэмнэ:
( link rel="icon" type="image/x-icon" href="/images/favicon.ico">)

Таб хуудасны гарчгийн зүүн талд доорх favicon зураг харагдана.
