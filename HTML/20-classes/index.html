HTML class attribute

HTML class шинж чанарыг HTML элементийн ангиллыг тодорхойлоход хэрэглэгддэг.

Хэд хэдэн HTML элементүүд ижил ангилалтай байж болно.

Using The class Attribute

Загварын хүснэгт дэх ангиллын нэрийг зааж өгөхийн тулд class шинж чанарыг ихэвчлэн ашигладаг.
Үүнийг мөн JavaScript ашиглан тухайн ангиллын нэр бүхий элементүүдэд хандаж, удирдах боломжтой.

Доорх жишээ city утгатай class шинж чанартай <div> элементтэй байна.
<div> элемент нь <head> хэсгийн .city загвараар тодорхойлогдсон байна:

<style>
.city {
background-color: honeydew;
color: dodgerblue;
border: 1px solid cornflowerblue;
margin: 20px;
padding: 20px;
}
</style>

<div class="city">
<h2>London/h2>
<p>London is the capital of England.</p>
</div>

Дараах жишээ нь note утгатай class шинж чанартай <span> элементтэй байна.
<span> элемент <head> хэсгийн .note загвараар тодорхойлогдсон байна:

<style>
.note {
font-size: 120%;
color: red;
}
</style>

<h1>My <span class="note">Important</span> Heading</h1>
<p>This is some <span class="note">important</span> text.</p>

note-тэй жишээг турших боломжгүй байв!

Зөвлөмж: Class шинж чанарыг ямар ч HTML элемент дээр ашиглаж болно.
Анхаарах: class жижиг үсгээр бичигдэнэ!

The Syntax For Class

Class үүсгэх: class-н өмнө цэгэн тэмдэгт (.) бичнэ. Дараа нь CSS шинж чанарыг {} хаалтанд тодорхойлно:

.city {
background-color: honeydew;
color: dodgerblue;
padding: 10px;
}

Multiple Classes

HTML элементүүд нь нэгээс олон ангид харьяалагддаг. Олон ангиудыг тодорхойлохын тулд ангийн нэрсийг зайгаар тусгаарлах
ж.нь: div class="city main">. Элементийг заасан ангиудын дагуу загварчлах болно.

Дараах жишээнд 3-н <h2> элемент city ангилалд хамаарна.
London текстийг center-т байрлуулж, main ангилалд багтаажээ.

<style>
.city {
background-color: honeydew;
color: dodgerblue;
padding: 10px;
}
.main {
text-align: center;
}
</style>
<h2 class="city main">London</h2>
<h2 class="city">Paris</h2>
<h2 class="city">Tokyo</h2>

Different Elements Can Share Same Class

Өөр өөр HTML элементүүд нь ижил ангийн нэрийг зааж болно.

Дараах жишээнд <h2>, <p> хоёулаа city ангиллыг зааж байгаа бөгөөд ижил хэв маягтай байна:

<style>
.city {
background-color: honeydew;
color: dodgerblue;
padding: 10px;
}
</style>

<h2 class="city">Paris</h2>
<p class="city">Paris is the capital of France.</p>

Use of The class Attribute in JavaScript

class-г мөн JavaScript-ээр тодорхой элементүүд дээр тодорхой ажлуудыг гүйцэтгэхэд ашиглаж болно.
