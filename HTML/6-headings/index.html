HTML Headings [h6]

HTML гарчиг нь вэб хуудсан дээр харуулахыг хүссэн гарчиг эсвэл дэд гарчиг юм.

HTML гарчиг нь < h1> - < h6> тагаар тодорхойлогдно.

Headings Are Important

Баримтын бүтцийг харуулахын тулд гарчгийг ашиглах нь чухал юм.

< h1> гарчигийг үндсэн гарчигт, < h2> дараагийн дэд гарчиг, дараа нь < h3> - < h6> жижиг гарчигуудад ашигладаг.

Bigger Headings

HTML гарчиг бүр үндсэн хэмжээтэй байна.

style attribute -ийн CSS font-size утгыг ашиглан гарчгаа дурын хэмжээгээр зааж өгч болно.

< h1 style="font-size:60px;">Heading< /h1> - гарчигийн хэмжээг зааж байна.
