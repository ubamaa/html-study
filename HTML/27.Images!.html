HTML images (Зураг оруулах)

Зураг нь вэб хуудасны дизайн болон харагдах байдлыг сайжруулж чадна.

HTML Images Syntax

HTML img таг нь вэб хуудсанд зураг оруулахад хэрэглэгддэг.

Шошго нь img шаардлагатай хоёр шинж чанартай:

src - Зурганд хүрэх замыг зааж өгнө.

alt - Зургийн өөр текстийг зааж өгнө.

Энэ img таг хоосон, зөвхөн шинж чанаруудыг агуулсан тул хаах таг байхгүй.

Syntax: (img src="url" alt="alternatetext")

The src Attribute

Шаардлагатай src атрибут нь зураг руу хүрэх замыг (URL) зааж өгдөг.

The alt Attribute

alt- HTML шинж чанар нь зургийг харуулах боломжгүй бол өөр текстийг харуулна.

Атрибутын alt утга нь зургийг тайлбарлах ёстой:

Хэрэв хөтөч зураг олж чадахгүй бол alt нь атрибутын утгыг харуулах болнo.

Image Size - Width and Height

width, height шинж чанарууд нь зургийн өргөн, өндрийг үргэлж пикселээр тодорхойлдог.

Зургийн өргөн ба өндрийг тодорхойлохын тулд style атрибутыг ашиглаж болно: style="width:500px;height:600px;

Эсвэл width, height шинж чанаруудыг ашиглаж болно: width="500" height="600"

Width and Height, or Style?

Width, height, style шинж чанарууд HTML дээр хүчинтэй.

Гэхдээ style шинж чанарыг ашиглахыг санал болгодог, учир нь хуудасны зургийн хэмжээг өөрчлөхөөс сэргийлнэ.

Images in Another Folder

Хэрэв танд дэд хавтас дотор зураг байгаа бол src шинж чанарт хавтасны нэрийг зааж өгөх ёстой.

Images on Another Server/Website

Өөр сервер дээрх зураг руу чиглүүлэхийн тулд src шинж чанарт үнэмлэхүй URL-г зааж өгдөг.

Animated Images (Хөдөлгөөнт зураг)

HTML нь хөдөлгөөнт GIF-г зөвшөөрдөг.

Image as a Link (Зургийг холбоос хэлбэрээр)

Зургийг холбоос болгон ашиглахын тулд a таг дотор img тагийг байрлуулна.

Зургийг текстийн баруун эсвэл зүүн талд хөвүүлэхийн тулд CSS float шинж чанарыг ашиглана.
