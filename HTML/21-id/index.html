HTML id Attribute

HTML < id> атрибут нь HTML элементийн өвөрмөц id-г тодорхойлоход хэрэглэгддэг.

HTML баримт бичигт ижил ID-тай нэгээс олон элемент байж болохгүй.

Using the id Attribute

< id> шинж чанар нь HTML элементийн өвөрмөц ID-г тодорхойлдог.

< id> шинж чанар нь загварын тодорхой хэв маягийг зааж өгөхөд ашиглагддаг.
id синтакс: Хэш тэмдэг (#) id нэр, дараа нь CSS шинж чанарыг {} хаалтанд тодорхойлно.

Дараах жишээнд myHeader нэртэй id, < h2> элементтэй байна.
< h2> элементийг < head> хэсгийн #myHeader загварын тодорхойлолтын дагуу загварчилсан байна:

style>
#myHeader {
background-color: lightblue;
color: black;
padding: 40px;
text-align: center;
}
< /style>

< h2 id="myHeader">Using the id Attribute/h2>

Note: < id> жижиг үсгийг мэдэрдэг! Мөн дор хаяж 1 тэмдэгт агуулсан байх ёстой,
тоогоор эхэлж болохгүй, хоосон зай (tab) агуулаагүй байх ёстой.

Difference Between Class and ID

class-г олон HTML элемент ашиглаж болно, < id>-г 1 хуудсанд зөвхөн 1 HTML элемент ашиглах ёстой.

Энэ хуудсанд оруулсан загварын жишээ:

style>
/* Style the element with the id "myHeader" */
#myHeader {
background-color: moccasin;
color: black;
padding: 40px;
text-align: center;
}

/* Style all elements with the class name "city" */
.city {
background-color: moccasin;
color: purple;
padding: 10px;
}
< /style>

!-- An element with a unique id -->
< h2 id="myHeader">Using The id Attribute in JavaScript/h2>

!-- Multiple elements with same class -->
< h2 class="city">Difference Between Class and ID/h2>
< h2 class="city">HTML Bookmarks with ID and Links/h2>

HTML Bookmarks with ID and Links

HTML bookmarks вэб хуудасны тодорхой хэсэг рүү шилжих боломжийг олгодог.

Хавчуурга ашиглахын тулд эхлээд үүнийг үүсгэж: < h2 id="C4">Chapter 4/h2>

Дараа нь түүнд холбоос нэмэх хэрэгтэй: < a href="#C4">Jump to Chapter 4/a>

Жишээг харах

Эсвэл өөр хуудаснаас хавчуургын холбоос ("Jump to Chapter 4") нэмж болно:
< a href="html_demo.html#C4">Jump to Chapter 4/a>
