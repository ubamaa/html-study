HTML Introduction

HTML бол вэб хуудас үүсгэх стандарт тэмдэглэгээний хэл юм.

What is HTML?

HTML нь Hyper Text Markup Language гэсэн үгийн товчлол юм.

HTML бол вэб хуудас үүсгэх стандарт тэмдэглэгээний хэл юм.

HTML нь вэб хуудасны бүтцийг тодорхойлдог.

A Simple HTML Document

<!DOCTYPE html> нь HTML5 баримт бичгийг тодорхойлдог.

<html> нь HTML хуудасны үндсэн элемент.

<head> нь HTML хуудасны мета мэдээллийг агуулдаг.

<title> нь HTML хуудасны гарчгийг тодорхойлдог (энэ нь хуудасны таб дээр харагддаг).

<body> нь document-н биеийг тодорхойлдог (гарчиг, мөр, зураг, гипер холбоос, хүснэгт, жагсаалт гэх мэт агуулгуудыг хадгалдаг).

<h1> нь эхний том гарчиг.

<p> нь догол мөрийг тодорхойлдог.

HTML элемент гэж юу вэ?

HTML элемент нь эхлэл тагаар эхлээд төгсгөлийн таг хүртэлх бүх зүйл юм.

HTML элемент нь start tag, content, end tag гэж тодорхойлогддог.
