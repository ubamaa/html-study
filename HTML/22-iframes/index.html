HTML Iframes

HTML iframe нь вэб хуудас доторх вэб хуудсыг харуулахад хэрэглэгддэг.

Швед хэл сурцгаая!

HTML Iframe Syntax

HTML <iframe> таг нь шугаман хүрээг тодорхойлдог.

Одоогийн HTML хуудаст өөр веб хуудсыг оруулахын тулд шугаман хүрээг ашигладаг.

Syntax: <iframe src="url" title="description">v/iframe>

Зөвлөгөө: <iframe>-д title шинж чанарыг үргэлж оруулахыг зөвлөж байна.
Энэ нь дэлгэц уншигч <iframe>-н агуулгыг уншихад ашигладаг.

Iframe - Set Height and Width

viframe>-н хэмжээг тодорхойлохын тулд width, height шинж чанаруудыг ашиглана:

<iframe src="url" height="200" width="300" title="description"></iframe>
(анхдагч өгөгдлөөр height, width пикселээр заасан байдаг).

Эсвэл style шинж чанарыг нэмж, CSS height, width ашиглаж болно:
<iframe src="url" style="height:200px;width:300px;" title="description"></iframe>

Iframe - Remove the Border

Анхдагч өгөгдлөөр <iframe> нь эргэн тойрондоо хүрээтэй байдаг.

iframe-н үндсэн хүрээг арилгахын тулд style шинж чанарыг нэмж, CSS border ашиглана:
<iframe src="url" style="border:none;" title="description"></iframe>

CSS-ийн тусламжтайгаар iframe хүрээний хэмжээ, хэв маяг, өнгийг өөрчлөх боломжтой:
<iframe src="url" style="border:2px solid red;" title="description"></iframe>

Iframe - Target for a Link

<Iframe>-г холбоосын зорилтот хүрээ болгон ашиглаж болно.

Холбоосын target шинж чанарыг iframe-н name шинж чанартай холбох ёстой:
(target атрибутын нэр iframe-н нэртэй таарч байх ёстой)

<iframe src="url" name="iframe_a" title="description"></iframe>
<p><a href="https://www.Internet Address"target="iframe_a">NAME</a></p>

Хуудасны эхэнд зорилтот холбоосын жишээг оруулсан!
