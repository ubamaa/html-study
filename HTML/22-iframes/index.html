Iframe - Target for a Link

Швед хэл сурцгаая!

HTML iframe нь вэб хуудас доторх вэб хуудсыг харуулахад хэрэглэгддэг. iframe таг нь шугаман хүрээг тодорхойлдог.

src attribute нь оруулах хуудасны URL-г тодорхойлдог.

iframe-н хэмжээг тодорхойлохын тулд height болон width шинж чанаруудыг ашиглана.

style шинж чанар нэмээд CSS height болон width шинж чанаруудыг ашиглаж болно.

CSS-ийн тусламжтайгаар iframe-н хүрээний хэмжээ, хэв маяг, өнгийг өөрчлөх боломжтой.
