HTML Lists (HTML жагсаалтууд)

HTML жагсаалт нь вэб хөгжүүлэгчдэд холбогдох элементүүдийг жагсаалт болгон бүлэглэх боломжийг олгодог.

  • HTML Lists
    • Unordered Lists (Эрэмбэлэгдээгүй жагсаалт)
    • Ordered Lists (Эрэмбэлэгдсэн жагсаалт)
    • Other lists (Бусад жагсаалт)

    Unordered HTML List

    Unordered List нь ul тагаар тодорхойлогдоно. Жагсаалтын зүйл бүр li тагааc эхэлнэ.

    Жагсаалтын анхдагч өгөгдөл (жижиг хар дугуй) тэмдэглэгээгээр тэмдэглэгддэг.

    Unordered HTML List - Choose List Item Marker

    CSS list-style-type шинж чанарыг жагсаалтын тэмдэглэгээг тодорхойлоход ашигладаг.
    Энэ нь дараах утгуудтай:

    • disc - Жагсаалтын жижиг хар дугуй тэмдэглэгээг тодорхойлно.
    • circle - Жагсаалтын тэмдэглэгээг тойрог болгож тохируулна.
    • square - Жагсаалтын тэмдэглэгээг дөрвөлжин болгож тохируулна.
    • none - Тэмдэглэгээгүй жагсаалт.
      • -Бас жагсаалт дотор жагсаалт оруулж болно.

      Nested HTML Lists

      Жагсаалтыг салаалж болно (жагсаалт доторх жагсаалт). Салбарласан жагсаалтыг li тагаар гүйцэтгэнэ.

      Horizontal List with CSS

      HTML жагсаалтыг CSS-ийн тусламжтайгаар янз бүрийн аргаар загварчилж болно.
      Нэг түгээмэл арга бол жагсаалтыг хэвтээ байдлаар загварчлах, навигацийн цэс үүсгэх явдал юм.

      Жагсаалтыг хэвтээ байдлаар харуулахын тулд float:left CSS шинж чанарыг ашиглана.

      Ordered HTML List

      Ordered Lists нь ol тагаар тодорхойлогдоно. Жагсаалтын зүйл бүр li тагаар эхэлнэ.

      Анхдагч өгөгдлөөр жагсаалтын тэмдэглэгээ тоон тэмдэглээтэй.

      Жишээг оруулах боломжгүй!

      Ordered HTML List - The Type Attribute

      ol тагийн type шинж чанар нь жагсаалтын тэмдэглэгээний төрлийг тодорхойлдог.

      1. type="1" - Жагсаалтын тэмдэглэгээг тоогоор дугаарлана.
      1. type="A" - Жагсаалтын тэмдэглэгээг том үсгээр дугаарлана.
      1. type="a" - Жагсаалтын тэмдэглэгээг жижиг үсгээр дугаарлана.
      1. type="I" - Жагсаалтын тэмдэглэгээг том ром тоогоор дугаарлана.
      1. type="i" Жагсаалтын тэмдэглэгээг жижиг ром тоогоор дугаарлана.
      1. start - Заасан тооноос дугаарлагдаж эхлэнэ.
        1. Мөн жагсаалт дотор жагсаалт оруулж болно.

        Control List Counting

        Анхдагчаар дугаарлагдсан жагсаалт 1-ээс тоолж эхэлдэг.
        Хэрэв та заасан тооноос тоолж эхлэхийг хүсвэл start шинж чанарыг ашиглана.

        Nested HTML Lists

        Жагсаалтыг үүрлүүлж болно (жагсаалт доторх жагсаалт).
        li таг нь шинэ жагсаалт болон зураг, холбоос гэх мэт бусад HTML элементүүдийг агуулдаг.

        Other Lists

        HTML нь мөн тайлбарын жагсаалтыг дэмждэг.

        HTML Description Lists

        Тайлбарын жагсаалт нь нэр томьёо бүрийн тайлбар бүхий жагсаалт юм.

        dl таг тайлбарын жагсаалтыг, dt таг нэр томьёог (name), dd таг нь тайлбар бүрийг тодорхойлдог.

        Жагсаалтанд нэр томьёо оруулан тайлбар нэмж болно.:
        - Тайлбарын жагсаалтыг тодорхойлохын тулд dl таг ашиглана.
        - Тайлбарын нэр томъёог тодорхойлоход dt тагийг ашиглана.
        - Нэр томъёог тайлбарлахын тулд dd таг ашиглагдана.
        HTML List Tags
        Tag Description
        ul Defines an unordered list Эрэмбэлэгдээгүй жагсаалтыг тодорхойлно.
        ol Defines an ordered list Эрэмбэлэгдсэн жагсаалтыг тодорхойлно.
        li Defines a list item Жагсаалтын элементийг тодорхойлно.
        dl Defines a description list Тайлбарын жагсаалтыг тодорхойлно.
        dt Defines a term in a description list Тайлбарын жагсаалтад нэр томьёо тодорхойлно.
        dd Describes the term in a description list Тайлбарын жагсаалтад байгаа нэр томъёог тайлбарлана.
