HTML Comments

HTML comment нь хөтөчид харагдахгүй, харин тэдгээр нь HTML эх кодыг баримтжуулахад тусална.

HTML Comment Tag

Доорх синтаксийг ашиглан HTML эх сурвалждаа тайлбар нэмж, бас нууж болно.

< !--сэтгэгдэл нэмэх хэсэг-- >

Эхлэлийн тагд анхаарлын тэмдэг (!) байгааг анхаарах, харин төгсгөлийн тэмдэглэгээнд байхгүй.

Add Comments

Comment-н тусламжтайгаар HTML коддоо мэдэгдэл, сануулагч байрлуулах боломжтой.

Энэ мөрийн дээр, доор бичсэн мөрүүд нуугдсан байгаа.

Hide Content

Бичсэн сэтгэгдлийн агуулгыг нуухад ашиглаж болно.

Доор бичигдсэн мөр нууцлагдсан байгаа.

Hide Inline Content
