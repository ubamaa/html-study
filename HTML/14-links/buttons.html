Link Buttons (Товчлуур холбоос)

Холбоосыг CSS ашиглан товчлуур хэлбэрээр мөн загварчилж болно.

Товчлуурын утгыг доор тайлбартай оруулав.

<style>

a:link, a:visited { background-color: greenyellow;} - холбоосын анхны болон зочилсоны дараах өнгө.

color: tomato; - товчлуурын текстийн өнгө.

padding: 10px 15px; - текст болон хүрээ хоорондын зай.

text-align: center; - товчлуурын текстийн байршил.

text-decoration: none; - товчлуурын текст доогуур зураасгүй байхгүй.

display: inline-block; -

a:hover, a:active { background-color: yellow;} - холбоост хүрэхэд, мөн дарахад солигдох өнгө.

</style>

Links файлтай танилцана уу!
