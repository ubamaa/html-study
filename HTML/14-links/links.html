HTML Links - Hyperlinks

HTML холбоосууд нь гипер холбоосууд юм.

Xолбоос дээр дараад өөр хуудас руу шилжих боломжтой.

HTML Links - Syntax

HTML <a> таг нь гипер холбоосыг тодорхойлдог.

Синтакс: (<a href="url">link text </a>)

<a> элементийн хамгийн чухал шинж чанар нь href бөгөөд холбоосын очих газрыг зааж өгдөг.

HTML Links - The target Attribute

Холбоотой хуудас нь тухайн хөтөчийн цонхонд харагдах болно. Үүнийг өөрчлөхийн тулд холбоосын өөр замыг зааж болно.

Аттрибут target нь холбосон хуудсыг хаана нээхийг зааж өгдөг.

target дараах утгуудын аль нэгтэй байж болно:

_self - Link-холбоос нь тухайн цонх/таб дээр нээнэ.

_blank - Баримтыг шинэ цонх/таб дээр нээнэ.

_parent - Баримтыг эх хүрээ дотор нээнэ.

_top - Баримтыг цонхны бүтэн хэсэгт нээнэ.

Absolute URLs vs. Relative URLs

Absolute URL (бүтэн вэб хаяг) нь href attribute ашиглаж байна.

Relative URL ("https://www" хэсэггүйгээр) нэг вэбсайт дахь хуудасны холбоосыг зааж өгнө.
