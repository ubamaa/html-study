HTML Background Images

Бүх HTML элементийн дэвсгэр зургийг зааж өгч болно.

Background Image on a HTML element

HTML элементэд дэвсгэр зураг нэмэхийн тулд HTML style attribute, CSS-н background-image шинж чанарыг ашиглана.

HTML элемент дээр дэвсгэр зураг нэмэхэд (p элемент):

(<p style="background-image: url('img_girl.jpg');">)

Мөн <head> хэсгийн style> элементэд дэвсгэр зургийг зааж өгч болно:

<style> p { background-image: url('img_girl.jpg'); } </style>

Background Image on a Page

Хэрэв хуудсыг бүхэлд нь дэвсгэр зурагтай болгохыг хүсвэл <body> элементэд дэвсгэр зургийг зааж өгөх ёстой:

<style> body { background-image: url('img_girl.jpg'); }</style>

Background Repeat

Хэрэв дэвсгэр зургийн хэмжээ нь жижиг бол зураг дэлгэц дүүргэхээр хэвтээ эсвэл босоо байдлаар давтагдана.

Дэвсгэр зургийг давтамжгүй болгохын тулд background-repeat шинж чанарыг давталтгүй болгож тохируулна.

<style>
body { background-image: url('example_img_girl.jpg');
background-repeat: no-repeat; }
</style>

Background Cover

Хэрэв дэвсгэр зургийг бүтэн дэлгэцэнд тавихыг хүсвэл background-size шинж чанарыг cover болгож тохируулна.

Мөн background-attachment шинж чанарыг fixed болгож тохируулна:

Ингэснээр дэвсгэр зураг нь суналтгүйгээр анхны харьцаагаа хадгалах болно.

<style>
body {
background-image: url('img_girl.jpg');
background-repeat: no-repeat;
background-attachment: fixed;
background-size: cover; }
</style>

Энэ дэд бүлгийн жишээг ашиглан хуудасны дэвсгэр зургийг оруулав.

Background Stretch

Дэвсгэр зургийг бүтэн дэлгэц болгохыг хүсвэл background-size шинж чанарыг 100% 100% болгож тохируулна.
Дэвсгэр зураг нь бүтэн дэлгэцийг хамрах боловч зураг сунасан дүрстэй байна.

<style>

body {
background-image: url('img_girl.jpg');
background-repeat: no-repeat;
background-attachment: fixed;
background-size: 100% 100%; }
</style>
