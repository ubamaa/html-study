HTML picture Element

HTML <picture> элемент нь янз бүрийн төхөөрөмж эсвэл дэлгэцийн хэмжээтэй зургуудыг харуулах боломжтой.

The HTML picture Element

<picture> элемент нь нэг буюу хэд хэдэн <source> элементийг агуулж байдаг.

<source> элемент тус бүр нь srcset атрибутаар дамжуулан өөр өөр зургийг илэрхийлдэг.

<source> элемент бүхэн media шинж чанартай байдаг, энэ нь тухайн зураг хэзээ хамгийн тохиромжтой болохыг тодорхойдог.

<picture>
<source media="(min-width: 650px)" srcset="images/360.jpg">
<source media="(min-width: 465px)" srcset="images/iPhone6s.jpg">
<img src="img_girl.jpg">
</picture>

<img> элементийг <picture> элементийн хэсэгт заавал дагалдуулна. <picture> элементийг дэмждэггүй хөтчүүд
эсвэл <source> тагуудын аль нь ч таарахгүй тохиолдолд <img> элементийг ашигладаг.

When to use the Picture Element

Элементийн (<picture>) 2 үндсэн зорилго байдаг:

1. Bandwidth (Дамжуулах зурвасын өргөн)
Хэрэв жижиг дэлгэц, төхөөрөмжтэй бол том зургийн файлыг татаж авах шаардлагагүй.
Хөтөч нь зохих шинж чанарын утгуудтай эхний source элементийг ашиглах болно.

Жишээ нь: Дээр оруулсан зургийн эхний зургийг программ тохиромжтой учир сонгосон байна.

2. Format Support (Форматын дэмжлэг)
Зарим хөтөч эсвэл төхөөрөмж нь бүх зургийн форматыг дэмждэггүй.
Бүх форматын зургийг нэмж болох бөгөөд <picture> элемент нь анхны дэмжигдсэн зургийн форматыг ашиглана.

<picture>
<source srcset="img_avatar.png">
<source srcset="img_girl.jpg">
<img src="img_beatles.gif" alt="Beatles" style="width:auto;">
</picture>

энэ жишээн дээрх эхний зураг буюу шарлуулсан утгыг программ сонгосон.
