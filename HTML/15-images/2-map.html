HTML Image Maps

HTML газрын зургийн (image maps) тусламжтайгаар зураг дээр товших боломжтой хэсгийг үүсгэж болно.

Image Maps

HTML map таг нь зургийн газрын зургийг тодорхойлдог.

Зургийн газрын зургийг товших талбайг нэг буюу хэд хэдэн area тагаар тодорхойлно.

Компьютер, гар утас эсвэл аяга кофе дээр дарж шинэ хуудас руу орж болно!

How Does it Work?

Зургийн газрын зураг үүсгэхийн тулд зураг дээр товших боломжтой хэсгийг тодорхойлсон HTML код хэрэгтэй.

The Image

Зургийг img таг ашиглан оруулна.

Бусад зургуудаас цорын ганц ялгаа нь usemap шинж чанарыг нэмэх ёстой.

(img src="image.jpg" alt="text" usemap="#workmap")

Usemap утга нь # хэштэгээр эхэлнэ.

Create Image Map

map элемент нь зургийн газрын зураг үүсгэхэд хэрэглэгддэг бөгөөд name шинж чанарыг ашиглан зурагтай холбодог.

(map name="workmap"/map)

Name шинж чанар нь img-ийн usemap шинж чанартай ижил утгатай байх ёстой.

The Areas:

Товших хэсгийг area элемент ашиглан тодорхойлно.

Товших хэсгийн хэлбэрийг тодорхойлох ёстой бөгөөд доорх утгуудын аль нэгийг сонгож болно:

rect - тэгш өнцөгт талбайг тодорхойлно.

circle - дугуй талбайг тодорхойлно.

poly - олон өнцөгт талбайг тодорхойлдог.

default - бүсийг бүхэлд нь тодорхойлдог.

- Shape="rect"

Shape="rect"-ийн координатууд нэг нь х тэнхлэгт, нөгөө нь у тэнхлэгт зориулагдсан болно.

(area shape="rect" coords="34, 44, 270, 350" href="Images/mac.jpg")

34,44 координатууд нь зүүн дээд захаас 34 пиксел, дээд талаас 44 пиксел зайд байрладаг.

270,350 координат нь баруун доод захаас 270 пиксел, дээд талаас 350 пиксел зайд байрладаг.

- Shape="circle"

Тойргийн талбай нэмэхийн тулд эхлээд тойргийн төвийн координатыг олно.

337,300 координат нь аягатай кофений тойргийн төв, 44пиксел тойргийн радиус.

(area shape="circle" coords="337, 300, 44" href="Images/koppkaffe.jpg")

- Shape="poly"

Шулуун шугмаас (олон өнцөгт) бүрдэх хэлбэрийг үүсгэдэг олон тооны координатын цэгүүдийг агуулдаг.
