CSS Comments

CSS comment-ууд нь хуудас дээр харагдахгүй ч баримтын эх кодыг боловсруулахад тусалж чадна.

CSS Comments

Comment нь кодыг тайлбарлахад хэрэглэгддэг бөгөөд эх кодыг дараа нь засварлахад тусална.

CSS Comment нь <style> element дотор байрладаг, (/*) эхэлж (*/) дуусна.

<head>
<style>
/* This is a single-line comment */
p {
color: red;
}
</style>
</head>

Comment-ийг кодын аль ч хэсэгт нэмж болно:

<head>
<style>
p {
color: red; /* Set text color to red */
}
</style>
</head>

Сэтгэгдэл нь олон мөрийг эзэлж болно:

<head>
<style>
/* This is
a multi-line
comment */

p {
color: red;
}
</style>
</head>

HTML and CSS Comments

HTML comment нь <!-- ... -- > синтакс ашиглан HTML эх кодод тайлбар нэмэх боломжтой.

Доорх жишээнд HTML болон CSS тайлбаруудын хослолыг ашигласныг харж байна.

<head>
<style>
p {
color: red; /* Set text color to red */
}
</style>
</head>
<body>

<h2>My Heading</h2>

<!-- These paragraphs will be red -->

</body>
