CSS HEX Colors

3 Digit HEX Value

3-н оронтой HEX код нь 6-н оронтой 16-тын тооллын системд үндэслэсэн кодуудын товчлол юм.

Гурван оронтой зургаан талт код нь дараах хэлбэртэй байна: #rgb
r, g, b нь 0-f хүртэлх утгатай red, green, blue өнгөнүүдийг илэрхийлнэ.

3-н оронтой 6-н талт кодыг бүx утгууд (RR, GG, BB) ижил үед л ашиглах боломжтой.

<head>
<style>
body {
background-color: #fc9; /* same as #ffcc99 */
}

h1 {
color: #f0f; /* same as #ff00ff */
}

p {
color: #b58; /* same as #bb5588 */
}
</style>
</head>

Хэрэв #ff00cc өнгөтэй бол дараах байдлаар бичиж болно: #f0c.

Дээрх жишээг хуудасны загварт ашигласан.
