CSS Shorthand Border Property

CSS Border - Shorthand Property

Кодыг богиносгохын тулд хилийн бүх салангид шинж чанаруудыг нэг шинж чанарт зааж өгч болно.

border шинж чанар нь дараах шинж чанаруудын товчлол юм:

  • border-width
  • border-style (зайлшгүй)
  • border-color

Жишээ:

border-width
border-style
border-color-үүдийг shorthand шинж чанарт багтаажээ.

border: 5px solid salmon;

Left Border

Зөвхөн нэг талын хилийн шинж чанарыг зааж өгч болно:

border-left-width
border-left-style
border-left-color шинж чанарууд товчлогдсон байна.

border-left: 5px solid slateblue;

Bottom Border

border-bottom-width
border-bottom-style
border-bottom-color шинж чанарууд shorthand шинж чанарт багтсан байна.

border-bottom: 2px solid darkorange;
