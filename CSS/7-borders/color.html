CSS Borders

CSS Border Color

border-color шинж чанарыг хүрээний өнгийг тохируулахад ашигладаг.

Хүрээний өнгийг дараах байдлаар тохируулж болно:

  • namn - өнгөний нэрийг зааж өгч болно.
  • HEX - HEX утгыг зааж өгөх "#ff0000" г. м.
  • RGB - RGB утгыг зааж өгөх "rgb(255,0,0)" г. м.
  • HSL - HSL утгыг зааж өгөх "hsl(0, 100%, 50%)" г. м.
  • transparent

Note: border-color нь дангаараа ашиглагдахгүй, border-style эхэнд тодорхойлогдоно.
Мөн хүрээний өнгийг тохируулаагүй бол тухайн элементийн өнгийг өвлөнө.

Хүрээ - border-style: solid; border-color: red;

Хүрээ - border-style: double; border-color: green;

Хүрээ - border-style: dotted; border-color: blue;

Specific Side Colors

border-color нь 4-н утгатай:

  • top border
  • right border
  • bottom border
  • left border

Тиймээс 4-н талын хилийг өөр өөр өнгөөр тодорхойлж болно.

Хүрээ - border-style: solid; border-color: red green blue yellow; (red top, green right, blue bottom, yellow left)

HEX Values

Хүрээний өнгийг зургаан оронтой (HEX) ашиглан тодорхойлж болно.

Хүрээ - border-style: solid; border-color: #6a5acd;

RGB Values

Хүрээний өнгийг RGB утгыг ашиглан зааж өгч болно:

Хүрээ - border-style: solid; border-color: rgb(238, 130, 238);

HSL Values

Мөн HSL утгыг ашиглаж болно.

Хүрээ - border-style: solid; border-color: hsl(0, 0%, 73%);
