CSS Border Sides

CSS Border - Individual Sides

Хүрээний тал бүрт өөр өөр хил зааж өгч болно.

CSS нь хил тус бүрийг (top, right, bottom, left) зааж өгөх шинж чанартай байдаг:

Хилийн 4-н шинж чанартай, 2 өөр загвар:
border-top-style: dotted;
border-right-style: solid;
border-bottom-style: dotted;
border-left-style: solid;

Дээрх жишээ үүнтэй ижил үр дүнтэй байна:
border-style: dotted solid;

Тэгэхээр, дээрх илэрхийллүүд хэрхэн ажилладагийн харцгаая.

Хэрэв border-style 4-н утгатай бол:

  • border-style: dotted solid double dashed;
    • top border - dotted
    • right border - solid
    • bottom border - double
    • left border - dashed

Хэрэв border-style 3-н утгатай бол:

  • border-style: dotted solid double;
    • top border - dotted
    • right and left - solid
    • bottom border - double

Хэрэв border-style нь 2 утгатай бол:

  • border-style: dotted solid;
    • top and bottom - dotted
    • right and left - solid

Хэрэв border-style нэг утгатай бол:

  • border-style: dotted;
    • all four borders - dotted

Мөн border-style-ийг border-width болон border-color шинж чанаруудтай хамт ашиглаж болно.
