CSS Borders

CSS border шинж чанар нь элементийн хүрээний хэв маяг, өргөн, өнгийг тодорхойлох боломжийг олгодог.

CSS Border Style

border-style шинж чанар нь ямар төрлийн хүрээ харуулахыг зааж өгдөг:

dotted - Цэгэн хүрээг тодорхойлно.

dashed - Тасархай зураасан хүрээг тодорхойлно.

solid - Зураасан хүрээг тодорхойлно.

double - Давхар зураастай хүрээг тодорхойлно.

groove - 3D ховилтой хүрээг тодорхойлно.

ridge - 3D товгор хүрээг тодорхойлно.

inset - 3D тасархай хүрээг тодорхойлно.

outset - 3D эхлэлийн хүрээг тодорхойлно.

none - Хүрээ байхгүй.

hidden - Далд хүрээг илтгэнэ.

mix - Холимог хүрээг харуулна.

border-style нь 4-н утгатай:

  • top border
  • right border
  • bottom border
  • left border

All CSS Border Properties
Property Description

border-bottom-color Доод хүрээний өнгийг тохируулна.
border-bottom-style Доод хүрээний хэв загварыг тохируулна.
border-bottom-width Доод хүрээний өргөнийг тохируулна.
border-color Хүрээний 4-н хилийн өнгийг тохируулна.
border-left Зүүн хилийн бүх шинж чанарыг 1 илэрхийлэлд тохируулна.

border-left-style Зүүн хүрээний хэв загварыг тохируулна.
border-left-width Зүүн хүрээний өргөнийг тохируулна.
