CSS Borders

CSS Border Width

border-width шинж чанар нь хүрээний 4-н хилийг тодорхойлдог.

Өргөнийг тодорхой хэмжээгээр (px, pt, cm, em г.м) эсвэл урьдчилан тодорхойлсон 3-н утгыг ашиглан тохируулж болно: thin, medium, thick.

Note: border-width шинж чанар нь ганцаараа ашиглагддаггүй. Эхэнд нь border-style шинж чанарыг заавал зааж өгнө.

Энэ хүрээ - solid, өргөн - 7px.

Энэ хүрээ - solid, өргөн - medium.

Энэ хүрээ - dashed, өргөн - thin.

Энэ хүрээ - dotted, өргөн - thick.

Энэ хүрээ - double, өргөн - 10px.

Энэ хүрээ - double, өргөн - 3px.
