CSS Layout - The position Property

position шинж чанар нь тухайн элементийн байрлал тогтоох аргын төрлийг (static, relative, fixed, absolute, sticky) тодорхойлдог.

The position Property

position шинж чанар нь тухайн элементэд ашиглах байршлын төрлийг тодорхойлдог.

5-н өөр байрлалын утга байдаг:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Эхлээд position шинж чанарыг тохируулах ёстой.
Дараа нь top, bottom, left, right шинж чанаруудыг ашиглан элементүүдийг байрлуулна.
Тэд байрлалын утгаас хамааран өөр өөрөөр ажилладаг.

position: static;

HTML элементүүд нь анхдагчаар static байдлаар байрладаг.
static байрлалтай элементүүдэд top, bottom, left, right шинж чанарууд нөлөөлдөггүй.
position: static тусгай байрлалгүй, хуудасны хэвийн урсгалын дагуу байрладаг!

<style>
div.static {
position: static;
border: 3px solid #73AD21;
}
</style>

<div class="static">This div element has position: static;</div>

Энд CSS ашигласан байна: position: static; байрлалтай div элемент.

position: relative;

position: relative;-тай элемент нь хэвийн байрлалтай буюу relative байрлалд байдаг.
Relative байрлалтай элементийн top, bottom, left, right шинж чанарыг тохируулах нь хэвийн байрлалаас нь холдуудаг.
Энэ нь элементээс үлдсэн хоосон зайг нөхөхийн тулд бусад контентыг тохируулахгүй.

<style>
div.relative {
position: relative;
left: 30px;
border: 3px solid #73AD21;
}
</style>

<div class="relative">This div element has position: relative;</div>

Энэ div элемент нь position: relative; байрлалтай.

position: fixed;

position: fixed; бүхий элемент нь хуудсанд үргэлж relative байдлаар харагдах бөгөөд хуудсыг гүйлгэсэн ч 1 байрандаа үлдэнэ гэсэн үг.
Элементийг байрлуулахад top, bottom, left, right шинж чанаруудыг ашигладаг.
fixed элемент нь хуудсанд ердийн байрлалтай байх үед байсан зайгаа үлдээдэггүй.

<style>
div.fixed {
position: fixed;
bottom: 0;
right: 0;
width: 300px;
border: 3px solid #73AD21;
}
</style>

<div class="fixed">This div element has position: fixed;</div>

div элемент position: fixed; байрлалтай.

Хуудасны баруун дээд буланд байрлах fixed байрлалтай элементийг харж байна.

position: absolute;

position: absolute-тэй элемент нь хамгийн ойр байрлах загвартай (relative байрлал дотор fixed шиг харагддаг) relative байрладаг.
Хэрэв absolut байрлалтай элемент нь давхар загваргүй бол баримтын үндсэн хуудсыг гүйлгэхэд хамтдаа хөдөлдөг.
Note: Absolute байрлалтай элементүүд нь хэвийн байрлал эзлэдэггүй бөгөөд элементүүдтэй давхцаж болно.

<style>
div.relative {
position: relative;
width: 400px;
height: 200px;
border: 3px solid #73AD21;
}

div.absolute {
position: absolute;
top: 80px;
right: 0;
width: 200px;
height: 100px;
border: 3px solid #73AD21;
}
</style>

<div class="relative">This div element has position: relative;
<div class="absolute">This div element has position: absolute;</div>
</div>

div элемент position: relative; байрлалтай.
div элемент position: absolute; байрлалтай.

position: sticky;

position: sticky; элемент нь хуудасный гүйлгэх байрлалд тулгуурлан байрлуулна.
Sticky элемент нь гүйлгэх байрлалаас хамааран relative болон fixed байрлалын хооронд шилжинэ.
Энэ нь хуудасны өгөгдсөн офсет байрлалд хүрэх хүртэл relative байрлалтай, дараа нь fixed шиг байрандаа үлддэг.

Note:Internet Explorer нь sticky байршлыг дэмждэггүй. Safari-д -webkit- угтвар шаардлагатай (доорх жишээг харах!).
Aжиллуулахын тулд sticky байрлалын top, bottom, right, left сонголтуудын аль нэгийг заах ёстой.
Энэ жишээнд sticky элемент нь гүйлгэх байрлалд хүрээд хуудасны дээд хэсэгт (top: 0;) наалдана.

<style>
div.sticky {
position: -webkit-sticky;
position: sticky;
top: 0;
padding: 5px;
background-color: #cae8ca;
border: 2px solid #4CAF50;
}
</style>

<div class="sticky">I am sticky!</div>

div элемент position: sticky; байрлалтай!


All CSS Positioning Properties

Property Description
bottom Байрлалын доод захын ирмэгийг тохируулна.
clip Absolute байрлалтай элементийг тайрахад ашиглана.
left Байрлалын зүүн захын ирмэгийг тохируулна.
position Элементийн байршлын төрлийг заана.
right Байрлалын баруун захын ирмэгийг тохируулна.
top Байрлалын дээд ирмэгийг тохируулна.
