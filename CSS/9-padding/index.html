CSS Padding

CSS padding шинж чанар нь тодорхой хил дотор элементийн агуулгын эргэн тойронд зай үүсгэхэд ашиглагддаг.

CSS-ийн тусламжтайгаар padding-ийг бүрэн хянах боломжтой.
Элементийн тал бүрт (top, right, bottom, left) дотор зайг тохируулах шинж чанарууд байдаг.

Padding - Individual Sides

CSS нь элементийн тал бүрийн зайг заадаг шинж чанартай:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Padding нь дараах утгуудтай байж болно.

  • length - px, pt, cm гэх мэт зайг зааж өгнө.
  • % - агуулж буй элементийн өргөний %-ийн дүүргэлтийг зааж өгнө.
  • inherit - зайг эх элементээс өвлөн авах ёстойг заана.

Padding - Shorthand Property

Кодыг богиносгохын тулд padding шинж чанарыг нэг шинж чанарт зааж өгч болно.

Padding шинж чанар нь дээр заасан бие даасан padding шинж чанаруудын товчлол шинж чанар юм.

Тэгэхээр, энэ нь хэрхэн ажилладаг вэ:

Хэрэв padding 4-н утгатай бол:

  • padding: 25px 50px 75px 100px;
    • top padding is 25px
    • right padding is 50px
    • bottom padding is 75px
    • left padding is 100px

Хэрэв padding 3-н утгатай бол:

  • padding: 25px 50px 75px;
    • top padding is 25px
    • right and left paddings are 50px
    • bottom padding is 75px

Хэрэв padding 2 утгатай бол:

  • padding: 25px 50px;
    • top and bottom paddings are 25px
    • right and left paddings are 50px

Хэрэв padding 1 утгатай бол:

  • padding: 25px;
    • all four paddings are 25px

Padding and Element Width

CSS width шинж чанар нь элементийн агуулгын талбайн өргөнийг заадаг.
Агуулгын талбай нь элементийн (хайрцагны загвар) дүүргэлт, хүрээ, ирмэгийн доторх хэсэг юм.

Хэрэв элемент заасан өргөнтэй бол тухайн элементийн padding-н хэмжээ элементийн нийт өргөнд нэмэгдэх болно.

Энд <div> элементэд 300px өргөн өгөгдсөн, padding 25px (баруун + зүүн талын зай = 50px),
ингээд <div> элементийн бодит өргөн нь 350px болж байна.

<style>
div.ex1 {
width: 300px;
background-color: yellow;
}

div.ex2 {
width: 300px;
padding: 25px;
background-color: lightblue;
}
</style>
<div class="ex1">This div is 300px wide.</div>
<div class="ex2">The width of this div is 350px, even though it is defined as 300px in the CSS.</div>

Padding-н хэмжээнээс үл хамааран элементийг заасан өргөнтэй нь байлгахын тулд box-sizing шинж чанарыг ашиглаж болно.
Энэ шинж чанар нь элементийн бодит өргөнийг хадгалахад хүргэдэг.

<style>
div.ex1 {
width: 300px;
background-color: yellow;
}

div.ex2 {
width: 300px;
padding: 25px;
box-sizing: border-box;
background-color: lightblue;
}
</style>


All CSS Padding Properties

Property Description
padding Бүх шинж чанарыг тохируулсан shorthand шинж чанар.
padding-bottom Элементийн доод зайг тохируулна.
padding-left Элементийн зүүн талын зайг тохируулна.
padding-right Элементийн баруун талын зайг тохируулна.
padding-top Элементийн дээд зайг тохируулна.
