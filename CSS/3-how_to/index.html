Three Ways to Insert CSS (CSS оруулах гурван арга)

Загварын хуудас оруулах гурван арга байдаг.

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

External CSS тусламжтайгаар бүхэл вэбсайтын харагдах байдлыг зөвхөн нэг файлаар өөрчлөх боломжтой!

HTML хуудас <head> хэсгийн <link> элементэд External style файлын холбоосыг агуулсан байх ёстой.
External style файлыг .css өргөтгөлөөр хадгална.

<head>
<link rel="stylesheet" href="mystyle.css">
</head>

Internal CSS

Хэрэв нэг HTML хуудсыг өвөрмөц хэв маягтай болгох бол Internal style-ийг ашиглаж болно.

Internal style-ийг <head> хэсгийн <style> элементэд тодорхойлно.

<style>
body {
background-color: linen;
}

h1 {
color: maroon;
margin-left: 40px;
}
</style>

Inline CSS

Нэг элементэд өвөрмөц загвар оруулахын тулд Inline style ашиглаж болно.

<p style="color:blue;text-align:center;"> Inline style ашиглахын тулд харгалзах элементэд загварын шинж чанарыг нэмнэ.

Multiple Style Sheets (Олон загварын хуудас)

Хэрэв нэг хуудсанд ижил selector (элемент)-үүдийг сонгосон бол хамгийн сүүлд бичсэн загварын утга үйлчилнэ.

Энэ жишээнүүд External style болон Internal style 2-ын хослол юм.

Жишээ 1.

External style хуудас нь <h1> элемент дараах загвартай: h1 {color: navy;}

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
<style>
h1 {
color: orange;
}
</style>
</head>

Internal style сүүлд тодорхойлогдсон тул <h1> элементүүд orange өнгөтэй болно.

Жишээ 2.

Жишээ 1.-н эсрэг тохиолдол:

<head>
<style>
h1 {
color: orange;
}
</style>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

Internal style-ийг External style-н өмнө тодорхойлсон бол <h1> элементүүд navy байх болно.

Cascading Order (Олон хэв маягийг нэг дор нэгтгэх)

HTML элементэд нэгээс олон загвар заасан тохиолдолд ямар хэв маягийг ашиглах вэ?

Бүх style нэг хуудсанд «каскад» болоход "эхнийх нь хамгийн давуу эрхтэй" гэсэн дүрмийн дагуу байх болно.

  1. Inline style (HTML элемент дотор)
  2. External ба Internal style sheets (head хэсэгт)
  3. Browser default (Үндсэн хөтөч)

Тиймээс inline style нь давуу эрхтэй бөгөөд External болон Internal styles, хөтчийн тохиргоог хүчингүй болгодог.
