CSS Flexbox

CSS Flexbox Layout Module

Flexbox Layout модулийг ашиглахаас өмнө байршлын 4-н горим байсан:

  • Вэб хуудасны хэсгүүдэд зориулсан Block
  • Текстийн хувьд Inline
  • Хоёр хэмжээст хүснэгтийн өгөгдөлд зориулсан хүснэгт
  • Элементийн тодорхой байрлалд зориулагдсан байрлал

Flexible Box Layout модуль нь float эсвэл тодорхой байрлалыг ашиглахгүйгээр уян хатан, хариу үйлдэл үзүүлэх байршлын бүтцийг боловсруулахад хялбар болгодог.

Flexbox Elements

Flexbox загварыг ашиглаж эхлэхийн тулд эхлээд уян хатан савыг тодорхойлох хэрэгтэй.
Flexible Layout нь нь dispaly шинж чанарыг flex гэж тохируулсан эх элементтэй байх ёстой.
Flex савны шууд child элементүүд автоматаар уян хатан зүйл болдог.

Доорх элемент нь 3-н уян хатан зүйл бүхий flex савыг (the blue area) илэрхийлнэ.

CSS Flex Container

Parent Element (Container)

Container гэдэг нь өмнө дурдсанчлан, 3-н уян зүйл (item) бүхий уян хатан сав (container) (цэнхэр хэсэг) юм.

Flex контейнер нь display шинж чанарыг flex болгож тохируулснаар уян хатан болдог.
flex container шинж чанарууд:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

The flex-direction Property

flex-direction шинж чанар нь сав нь уян хатан зүйлсийг аль чиглэлд байршуулахыг хүсч байгааг тодорхойлдог.

Column утга нь уян хатан зүйлсийг босоо байдлаар (from top to bottom) байршуулсныг зааж байна :

.flex-container {
display: flex;
flex-direction: column;
}

column-reverse утга нь уян хатан зүйлсийг босоо байдлаар (from bottom to top) байршуулна.

.flex-container {
display: flex;
flex-direction: column-reverse;
}

row утга нь уян хатан зүйлсийг хэвтээ байдлаар байрлуулна (from left to right).

.flex-container {
display: flex;
flex-direction: row;
}

row-reverse утга нь уян хатан зүйлсийг хэвтээ байдлаар (from right to left) байрлуулна.

.flex-container {
display: flex;
flex-direction: row-reverse;
}

}

The flex-flow Property

flex-flow шинж чанар нь flex-direction болон flex-wrap шинж чанаруудыг тохируулсан shorthand шинж чанар юм.

.flex-container {
display: flex;
flex-flow: row wrap;
}

flex-start утга нь уян хатан зүйлсийг савны эхлэлтэй зэрэгцүүлдэг (өгөгдмөл).

.flex-container {
display: flex;

}

.flex-container {

}

Perfect Centering

Дараах жишээнд маш нийтлэг хэв загварын асуудлыг шийдэх болно: perfect centering.

SOLUTION. justify-content болон align-items шинж чанаруудыг center гэж тохируулбал уян хатан зүйл perfect centering болно:

.flex-container {
display: flex;
height: 300px;
justify-content: center;
align-items: center;
}

The CSS Flexbox Container Properties

Property Description
align-content Flex-wrap шинж чанарыг өөрчилдөг. Энэ нь align-items-тэй төстэй, гэвч уян хатан зүйлсийг зэрэгцүүлдэггүй уян хатан шугмуудыг зэрэгцүүлдэг.
align-items Элементүүд нь хөндлөн тэнхлэгт байгаа зайг ашиглахын тулд уян зүйлсийг босоо байдлаар тэгшлэнэ.
display HTML элементэд ашиглагдах дэлгэцийн төрлийг заана.
flex-direction Flex контейнер доторх уян хатан зүйлсийн чиглэлийг заана.
flex-flow Flex-direction болон flex-wrap-д зориулсан shorthand шинж чанар юм.
flex-wrap Нэг мөрөнд хангалттай зай байхгүй бол уян хатан зүйлсийг ороох эсэхийг заана.
justify-content Үндсэн тэнхлэгт байгаа зайг ашиглахгүй байх үед уян хатан зүйлсийг хэвтээ байдлаар байрлуулна.
