CSS Flex Items

Child Elements (Items)

Flex контейнерийн шууд child элементүүд нь автоматаар flex элемент (item) болдог.

Дээрх элемент нь бүдэг ягаан өнгөтэй уян контейнерийн доторх 4-н ногоон уян элементийг төлөөлдөг.

<div class="flex-container">
<div>1</div>
<div>2</div>
<div>3</div>
<div>4</div>
</div>

Уян элементийн шинж чанарууд нь:

  • order
  • flex-grow
  • flex-shrink
  • flex-basic
  • flex
  • align-self

The order Property

order шинж чанар нь уян элементийн дарааллыг тодорхойлдог.
order-ийн утга нь тоо байх ёстой, анхдагч утга нь 0 юм.
Кодын эхний уян элементийг байршлын эхний элемент болгон харуулах албагүй.
Учир нь, order шинж чанарыг ашиглан flex-элементийн дарааллыг өөрчилж болно.

<div class="flex-container">
<div style="order: 3">1</div>
<div style="order: 2">2</div>
<div style="order: 4">3</div>
<div style="order: 1">4</div>
</div>

flex-row Property

flex-grow шинж чанар нь 1 flex-элемент нь бусад flex-элементүүдтэй харьцуулахад хэр хэмжээгээр өсөхийг тодорхойлдог.
Утга нь тоо байх ёстой, анхдагч утга нь 0 байна.

3 дахь уян элемент нь бусад уян элементүүдээсээ 8 дахин томорсон байна.

<div class="flex-container">
<div style="flex-grow: 1">1</div>
<div style="flex-grow: 1">2</div>
<div style="flex-grow: 8">3</div>
</div>

The flex-shrink Property

flex-shrink шинж чанар нь 1 уян элемент бусад уян элементүүдтэй харьцуулахад хэр хэмжээгээр агшихыг тодорхойлдог.
Утга нь тоо байх ёстой, анхдагч утга нь 1 байна.

9 дахь уян элементийг бусад уян элементийн адил багасгаагүй байгаа:

<div class="flex-container">
<div>1</div>
<div>2</div>
<div style="flex-shrink: 0">3</div>
<div>4</div>
<div>5</div>
<div>6</div>

The flex-basis Property

flex-basis шинж чанар нь уян элементийн анхны уртыг тодорхойлдог.

3 дахь уян элементийн анхны уртыг 200px болгож тохируулсан:

<div class="flex-container">
<div>1</div>
<div>2</div>
<div style="flex-basis: 200px">3</div>
<div>4</div>
</div>

The flex Property

flex шинж чанар нь flex-grow, flex-shrink болон flex-basis шинж чанаруудын товчилсон шинж чанар юм.

1 дахь уян элементийг flex-grow: 0, flex-shrink: 0, flex-basis: 200px болгоно:

<div class="flex-container">
<div style="flex: 0 0 200px">1</div>
<div>2</div>
<div>3</div>
<div>4</div>

The align-self Property

align-self шинж чанар нь уян контейнер доторх сонгосон элементийн зэрэгцүүлэлтийг тодорхойлдог.
align-self шинж чанар нь контейнерийн align-items шинж чанараар тохируулсан өгөгдмөл зэрэгцүүлэлтийг хүчингүй болгодог.

Эдгээр жишээн дээр align-self шинж чанарыг илүү сайн харуулахын тулд 200px өндөртэй контейнерыг ашигласан.

Сонгосон 3 дахь уян элементийг align-self: center; болгож контейнерийн голд зэрэгцүүлнэ:

<div class="flex-container">
<div>1</div>
<div>2</div>
<div style="align-self: center">3</div>
<div>4</div>
</div>

The CSS Flexbox Items Properties

Property Description
align-self Flex элементийн зэрэгцүүлэлтийг заана (flex container-ийн align-items шинж чанарыг дарна).
flex flex-grow, flex-shrink, болон the flex-basis шинж чанаруудын товчилсон шинж чанар.
flex-basis Уян элементийн анхны уртыг заана.
flex-grow Нэг контейнерт байгаа бусад уян элементтэй харьцуулахад 1 уян элемент хэр их өсөхийг заана.
flex-shrink Нэг контейнерт байгаа бусад уян хатан элементтэй харьцуулахад 1 уян элемент хэр хэмжээгээр агшихыг зааж өгнө.
order Нэг контейнер доторх уян хатан элементийн дарааллыг заана.
