Responsive Website using Flexbox

Flexbox ашиглан уян навигацийн талбар, уян контент бүхий хариу үйлдэл үзүүлэх вэб сайтыг үүсгэнэ.

Хариу өгөх эффектийг харахын тулд хөтчийн цонхны хэмжээг өөрчилж болно.
