CSS Flex Responsive

Responsive Flexbox

Хэрэв дэлгэцийн хэмжээтэй хоёр багана, жижиг дэлгэцийн хэмжээтэй (утас, таблет гэх мэт) нэг баганатай модель үүсгэхийг хүсвэл flex-direction шинж чанарыг row to column болгон уян чиглэлийг өөрчилж болно. (доорх жишээнд 800px):

Дараах жишээ нь дэлгэцийн хэмжээ 800px буюу түүнээс бага өргөнтэй үед уян элементүүдийг босоо (дээрээс доош) байдлаар (flex-direction: row;),
хэрэв дэлгэцийн хэмжээ 800px-ээс том бол уян элементүүдийг хэвтээ байдлаар (зүүнээс баруун тийш) flex-direction: row; болгож өөрчилнө:

.flex-container {
display: flex;
flex-direction: row;
}

/* Responsive layout - makes a one column layout instead of a two-column layout */
@media (max-width: 800px) {
.flex-container {
flex-direction: column; }
}

Өөр нэг арга бол уян элементийн flex шинж чанарын хувь хэмжээг өөрчлөж, янз бүрийн дэлгэцийн хэмжээтэй өөр байршил үүсгэх явдал юм.
Энэ жишээг ажиллуулахын тулд уян контейнерт flex-wrap: wrap;-г оруулах хэрэгтэйг анхаар:

.flex-container {
display: flex;
flex-wrap: wrap;
}

.flex-item-left {
flex: 50%;
}

.flex-item-right {
flex: 50%;
}

/* Responsive layout - makes a one column layout instead of a two-column layout */
@media (max-width: 800px) {
.flex-item-right, .flex-item-left {
flex: 100%; }
}
