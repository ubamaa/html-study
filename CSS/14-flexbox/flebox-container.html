CSS Flexbox

CSS Flexbox Layout Module

Flexbox Layout модулийг ашиглахаас өмнө байршлын 4-н горим байсан:

  • Вэб хуудасны хэсгүүдэд зориулсан Block
  • Текстийн хувьд Inline
  • Хоёр хэмжээст хүснэгтийн өгөгдөлд зориулсан хүснэгт
  • Элементийн тодорхой байрлалд зориулагдсан байрлал

Flexible Box Layout модуль нь float эсвэл тодорхой байрлалыг ашиглахгүйгээр уян хатан, хариу үйлдэл үзүүлэх байршлын бүтцийг боловсруулахад хялбар болгодог.

Flexbox Elements

Flexbox загварыг ашиглаж эхлэхийн тулд эхлээд уян хатан контейнерийг тодорхойлох хэрэгтэй.
Flexible Layout нь нь dispaly шинж чанарыг flex гэж тохируулсан эх элементтэй байх ёстой.
Flex контейнерийн шууд child элементүүд автоматаар уян элемент болно.

CSS Flex Container

Parent Element (Container)

Container гэдэг нь өмнө дурдсанчлан, 3-н уян элемент (item) бүхий уян сав (container) (ногоон хэсэг) юм.

Flex контейнер нь display шинж чанарыг flex болгож тохируулснаар уян хатан болдог.
flex container шинж чанарууд:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

The flex-direction Property

flex-direction шинж чанар контейнер нь уян элементийг аль чиглэлд байршуулахыг хүсч байгааг тодорхойлдог.

Column утга нь уян элементийг босоо байдлаар (from top to bottom) байршуулсныг зааж байна:

.flex-container {
display: flex;
flex-direction: column;
}

Perfect Centering

Дараах жишээнд маш нийтлэг хэв загварын асуудлыг шийдэх болно: perfect centering.

SOLUTION. justify-content болон align-items шинж чанаруудыг center гэж тохируулбал уян элемент perfect centering болно:

.flex-container {
display: flex;
height: 300px;
justify-content: center;
align-items: center;
}

The CSS Flexbox Container Properties

Property Description
align-content Flex-wrap шинж чанарыг өөрчилдөг. Энэ нь align-items-тэй төстэй, гэвч уян элементийг зэрэгцүүлдэггүй уян шугмуудыг зэрэгцүүлдэг.
align-items Элементүүд нь хөндлөн тэнхлэгт байгаа зайг ашиглахын тулд уян элементийг босоо байдлаар тэгшлэнэ.
display HTML элементэд ашиглагдах дэлгэцийн төрлийг заана.
flex-direction Flex контейнер доторх уян элементийн чиглэлийг заана.
flex-flow Flex-direction болон flex-wrap-д зориулсан shorthand шинж чанар юм.
flex-wrap Нэг мөрөнд хангалттай зай байхгүй бол уян хатан элементийг ороох эсэхийг заана.
justify-content Үндсэн тэнхлэгт байгаа зайг ашиглахгүй байх үед уян элементийг хэвтээ байдлаар байрлуулна.
