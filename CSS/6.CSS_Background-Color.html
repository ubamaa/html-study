CSS Backgrounds

CSS background шинж чанар нь элементүүдэд дэвсгэр эффект нэмэхэд ашигладаг.

CSS background дараах шинж чанаруудтай:

    background-color
    background-image
    background-repeat
    background-attachment
    background-position
    background(shorthand property)

CSS background-color

background-color шинж чанар нь элементийн дэвсгэр өнгийг тодорхойлдог.

Энд h1, p, div элементүүд өөр өөр дэвсгэр өнгөтэй байна.
