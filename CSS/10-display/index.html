CSS Layout - The display Property

Display шинж чанар нь байршлыг хянах хамгийн чухал CSS шинж чанар юм.

The display Property

Display шинж чанар нь тухайн элементийг хэрхэн харуулахыг заадаг.

HTML элемент бүр элементийн төрлөөс хамааран өгөгдмөл дэлгэцийн утгатай байна.
Ихэнх элементийн дэлгэцийн өгөгдмөл утга нь block эсвэл inline юм.

Block-level Elements

Block-level нь үргэлж шинэ мөрөнд эхэлдэг бөгөөд боломжтой өргөнийг эзэлдэг (баруун, зүүн тийш сунадаг).

Block түвшний элементүүдийн жишээ:

<div>
<h1> - <h6>
<p>
<table>
<dd>
<dt>
<hr>
<pre>
<ol>
<li>
<ul>
<form>
<header>
<footer>
<main>

Inline Elements

Inline элемент нь шинэ мөрөнд эхэлдэггүй бөгөөд зөвхөн шаардлагатай хэмжээгээр өргөнийг эзэлдэг.

Inline элементүүдийн жишээ:

<span> <a> <img> <button> <sup> <abbr> <cite> <script> <b> <map> <small> <bdo> <em> <i> <strong> <q> <sub> <br>

Display: none;

display:none нь JavaScript-д элементүүдийг устгах, дахин үүсгэхгүйгээр нуух, харуулах зорилгоор ихэвчлэн хэрэглэгддэг.

<script> элемент нь display: none;-г анхдагч байдлаар ашигладаг.

Override The Default Display Value

Дээр дурдсанчлан, элемент бүр өгөгдмөл дэлгэцийн утгатай байдаг. Гэсэн хэдий ч үүнийг дарж болно.

Inline элементийг Block элементээр солих (эсвэл эсрэгээр) нь вэб стандартад нийцсэн хуудсыг онцгой харагдуулахад тустай.

Жишээ 1.

Нийтлэг жишээ бол хэвтээ цэсэнд Inline <li> элементүүдийг үүсгэх явдал юм.
Холбоосуудын жагсаалтыг хэвтээ цэс хэлбэрээр харуулах:

  • HTML
  • CSS
  • JavaScript

<style>
li {
display: inline;
}
</style>

Note: Элементийн displays шинж чанарыг тохируулах нь элементийн төрлийг БИШ зөвхөн тухайн элементийг хэрхэн харуулахыг өөрчилдөг.
Тэгэхээр, display:none бүхий Inline элемент дотор бусад Block элемент байхыг хориглоно.

Жишээ 2.

Дараах жишээнд <span> элементүүдийг Block элемент болгон үзүүлэв:

<style>
span {
display: block;
}
</style>

A display property witha value of "block" results in a line break between each span elements.

Жишээ 3.

Дараах жишээнд <a> элементүүдийг Block элемент болгон үзүүлэв:

<style>
a {
display: block;
}
</style>

Hide an Element - display:none or visibility:hidden?

Жишээ 1.

display: none; гэж тохируулснаар элементийг нууж болно. Элемент нуугдаж, хуудас нь элемент байхгүй мэт харагдах болно:

<style>
h1.hidden {
display: none;
}
</style>

<h1 class="hidden">This is a hidden heading</h1>
<p>Notice that the h1 element with display: none; does not take up any space.</p>

Жишээ 2.

visibility:hidden; мөн элементийг нуудаг. Элемент нуугдах боловч байршилд нөлөөлж зайгаа хадгалсан хэвээр байна.

<style>
h1.hidden {
visibility: hidden;
}
</style>

<h1 class="hidden">This is a hidden heading</h1>
<p>Notice that the hidden heading still takes up space.</p>


CSS Display/Visibility Properties

Property Description
display Элементийг хэрхэн харуулахыг зааж өгнө.
visibility Элемент харагдах эсэхийг зааж өгнө.
