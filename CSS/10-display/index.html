CSS Layout - The display Property

Display шинж чанар нь байршлыг хянах хамгийн чухал CSS шинж чанар юм.

The display Property

Display шинж чанар нь тухайн элементийг хэрхэн харуулахыг заадаг.

HTML элемент бүр элементийн төрлөөс хамааран өгөгдмөл дэлгэцийн утгатай байна.
Ихэнх элементийн дэлгэцийн өгөгдмөл утга нь block эсвэл inline юм.

Block-level Elements

Block-level нь үргэлж шинэ мөрөнд эхэлдэг бөгөөд боломжтой өргөнийг эзэлдэг (баруун, зүүн тийш сунадаг).

Block-level элементүүдийн жишээ:

<div> - HTML баримт бичигт хуваагдал эсвэл хэсгийг тодорхойлдог.
<h1> - <h6> - HTML гарчгийг тодорхойлдог.
<p> - догол мөрийг тодорхойлдог.
<table> - HTML хүснэгтийг тодорхойлдог.
<dd> - тайлбарын жагсаалт дахь нэр/томъёог тодорхойлоход хэрэглэгддэг.
<dt> - тайлбарын жагсаалт дахь нэр/томъёог тодорхойлоход хэрэглэгддэг.
<hr> - HTML хуудасны сэдэвчилсэн завсарлагыг тодорхойлдог.
<pre> - урьдчилан форматласан текстийг тодорхойлдог.
<ol> - эрэмбэлэгдсэн жагсаалтыг тодорхойлдог.
<li> - жагсаалтын элементийг тодорхойлдог.
<ul> - эрэмбэлэгдээгүй жагсаалтыг тодорхойлно.
<form> - хэрэглэгчийн оруулах HTML хэлбэрийг бий болгоход ашигладаг.
<header> - танилцуулах контент эсвэл навигацийн холбоосуудын багц юм.
<footer> - баримт бичгийн төгсгөл хэсгийг тодорхойлдог.
<main> - баримт бичгийн үндсэн агуулгыг тодорхойлдог.
<address> - холбоо барих мэдээлэл, имэйл хаяг, URL, утасны дугаар, нийгмийн сүлжээний нэр гэх мэт нийтлэлийн зохиогч/эзэмшигчтэй холбоотой мэдээллийг тодорхойлдог.

Inline Elements

Inline элемент нь шинэ мөрөнд эхэлдэггүй бөгөөд зөвхөн шаардлагатай хэмжээгээр өргөнийг эзэлдэг.

Inline элементүүдийн жишээ:

<span> - текстийн хэсгийг тэмдэглэхэд ашигладаг.

<a> - 1 хуудаснаас нөгөө хуудас руу холбоход хэрэглэгддэг гипер холбоосыг тодорхойлдог.

<img> - HTML хуудсанд зураг оруулахад ашигладаг.

<button> - дарж болох товчлуурыг тодорхойлдог.

<sup> - текстийн дээд тэмдгийг тодорхойлдог.

<abbr> - товчлолыг тодорхойлдог.

<cite> - бүтээлч болон бүтээлийн нэрийг (ном, шүлэг, дуу, кино, уран зураг, баримал гэх мэт) тодорхойлдог.

<script> - клиент талын скриптийг (JavaScript) оруулахад ашигладаг.

<b> - ямар ч нэмэлт ач холбогдолгүйгээр тод бичвэрийг онцолж өгдөг.

<map> - зургийн газрын зургийг тодорхойлоход ашигладаг.

<small> - жижиг текстийг (зохиогчийн эрх болон бусад тайлбар гэх мэт) тодорхойлдог.

<bdo> - текстийн чиглэлийг өөрчлөхөд хэрэглэгддэг.

<em> - сонгосон текстийг тодруулахад ашигладаг.

<i> - өөр дуу хоолой, сэтгэл хөдлөл бүхий текстийн хэсгийг тодорхойлдог.

<strong> - өндөр ач холбогдолтой текстийг тодорхойлоход хэрэглэгддэг.

<q> - богино ишлэлийг тодорхойлдог.

<sub> - дэд тэмдэгтийн текстийг тодорхойлдог.

<br> - нэг мөрийн завсарлага оруулдаг.

<var> - програмчлал эсвэл математик илэрхийлэлд хувьсагчийг тодорхойлоход хэрэглэгддэг.

<output> - тооцооллын үр дүнг илэрхийлэхэд хэрэглэгддэг.

<time> - тодорхой цагийг (эсвэл огноо/цаг) тодорхойлдог.

<select> - нээгддэг жагсаалт үүсгэхэд ашигладаг.

Display: none;

display:none нь JavaScript-д элементүүдийг устгах, дахин үүсгэхгүйгээр нуух, харуулах зорилгоор ихэвчлэн хэрэглэгддэг.

<script> элемент нь display: none;-г анхдагч байдлаар ашигладаг.

Override The Default Display Value

Дээр дурдсанчлан, элемент бүр өгөгдмөл дэлгэцийн утгатай байдаг. Гэсэн хэдий ч үүнийг дарж болно.

Inline элементийг Block элементээр солих (эсвэл эсрэгээр) нь вэб стандартад нийцсэн хуудсыг онцгой харагдуулахад тустай.

Жишээ 1.

Нийтлэг жишээ бол хэвтээ цэсэнд Inline <li> элементүүдийг үүсгэх явдал юм.
Холбоосуудын жагсаалтыг хэвтээ цэс хэлбэрээр харуулах:

<style>
li {
display: inline;
}
</style>

Note: Элементийн displays шинж чанарыг тохируулах нь элементийн төрлийг БИШ зөвхөн тухайн элементийг хэрхэн харуулахыг өөрчилдөг.
Тэгэхээр, display:none бүхий Inline элемент дотор бусад Block элемент байхыг хориглоно.

Жишээ 2.

Дараах жишээнд <span> элементүүдийг Block элемент болгон үзүүлэв:

<style>
span {
display: block;
}
</style>

A display property witha value of "block" results ina line break between each span elements.

Жишээ 3.

Дараах жишээнд <a> элементүүдийг Block элемент болгон үзүүлэв:

<style>
a {
display: block;
}
</style>

Hide an Element - display:none or visibility:hidden?

Жишээ 1.

display: none; гэж тохируулснаар элементийг нууж болно. Элемент нуугдаж, хуудас нь элемент байхгүй мэт харагдах болно:

<style>
h1.hidden {
display: none;
}
</style>

<h1 class="hidden">This is a hidden heading</h1>
<p>Notice that the h1 element with display: none; does not take up any space.</p>

Жишээ 2.

visibility:hidden; мөн элементийг нуудаг. Элемент нуугдах боловч байршилд нөлөөлж зайгаа хадгалсан хэвээр байна.

<style>
h1.hidden {
visibility: hidden;
}
</style>

<h1 class="hidden">This is a hidden heading</h1>
<p>Notice that the hidden heading still takes up space.</p>


CSS Display/Visibility Properties

Property Description
display Элементийг хэрхэн харуулахыг зааж өгнө.
visibility Элемент харагдах эсэхийг зааж өгнө.
