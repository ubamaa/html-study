CSS Grid Container

Grid Container

HTML элементийг сүлжээний контейнер шиг ажлуулахын тулд display шинж чанарыг grid эсвэл inline-grid болгон тохируулах ёстой.
Grid контейнерууд нь багана, эгнээ болгон байрлуулсан grid item-үүдээс бүрдэнэ.

The grid-template-columns Property

grid-template-columns шинж чанар нь сүлжээний байршлын баганын тоог, мөн багана бүрийн өргөнийг тодорхойлдог.
Утга тус бүр нь харгалзах баганын өргөнийг зааж, зайгаар тусгаарлагдсан байдаг.
Хэрэв сүлжээний бүтцийг 4 баганатай болгохыг хүсвэл 4 баганын өргөнийг зааж өгнө,
хэрэв баганын өргөн бүгд адил хэмжээтэй бол auto гэж заана.

4 багана бүхий сүлжээ үүсгэх:

.grid-container {
display: grid;
grid-template-columns: auto auto auto auto;
}

Note: Хэрэв 4 баганатай сүлжээнд 4-өөс олон grid item байвал сүлжээ нь шинэ мөрийг автоматаар нэмж оруулна.

Мөн grid-template-columns шинж чанарыг баганын хэмжээг (өргөн) тодорхойлж болно.

4-н өөр хэмжээтэй баганууд үүсгэх:

.grid-container {
display: grid;
grid-template-columns: 80px 200px auto 40px;
}

grid-template-rows Property

grid-template-rows шинж чанар нь мөр бүрийн өндрийг тодорхойлдог.
Утга нь мөр бүрийн хэмжээ (өндөр)-ийг зааж, зайгаар тусгаарлагдсан байна.

.grid-container {
display: grid;
grid-template-rows: 80px 200px;
}

The justify-content Property

justify-content шинж чанар нь контейнерийн доторх сүлжээг бүхэлд нь зэрэгцүүлэхэд ашиглагддаг.
Note: Сүлжээний нийт өргөн нь контейнерийн өргөнөөс бага байж justify-content шинж чанар нь нөлөө үзүүлнэ.

space-evenly утга нь багануудын хооронд болон эргэн тойронд нь ижил зайтай болгоно.

.grid-container {
display: grid;
justify-content: space-evenly;
}

The align-content Property

align-content шинж чанар нь контейнерийн доторх бүхэл сүлжээг босоо байдлаар байрлуулахад ашиглагддаг.
Сүлжээний нийт өндөр нь контейнерийн өндрөөс бага байх ёстой, энэ нь align-content нөлөө үзүүлэх боломжтой.

center утга нь контейнерийн голд мөрүүдийг тэгшлэнэ:

.grid-container {
display: grid;
height: 400px;
align-content: center;
}
