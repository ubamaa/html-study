CSS Grid Item

Child Elements (Items)

Сүлжээний контейнер нь сүлжээний зүйлсийг (item) агуулдаг.

Анхдагч өгөгдлөөр, контейнер нь мөр болгоны багана тус бүрт нэг сүлжээний зүйлтэй (item) байдаг,
хэрэв олон багана эсвэл мөртөй болгох бол сүлжээний зүйлсийг загварчилж болно.

The grid-column Property:

grid-column шинж чанар нь тухайн item-ийг аль баганануудад байрлуулахыг тодорхойлдог.
Мөн grid-column шинж чанар нь grid-column-start, grid-column-end шинж чанаруудын товчилсон шинж чанар юм.

Тухайн item хаанаас эхлэх, хаана дуусахыг тодорхойлох ёстой.
Доорх жишээ: item1 1-р баганаас эхэлж 5-р баганаас өмнө дуусна:

.item1 {
grid-column: 1 / 5;
}

The grid-row Property:

grid-row шинж чанар нь item-ийг аль мөрөнд байрлуулахыг тодорхойлдог.
Мөн grid-row шинж чанар нь grid-row-start болон grid-row-end шинж чанаруудын товчилсон шинж чанар юм.

item хаана эхлэх, хаана дуусахыг тодорхойлох ёстой.
item1-ийг 1-р мөрнөөс эхэлж, 4-р мөрөнд төгсгөнө:

.item1 {
grid-row: 1 / 4;

The grid-area Property

grid-area шинж чанарыг grid-row-start, grid-column-start, grid-row-end, grid-column-end шинж чанаруудын товчилсон шинж чанар болгон ашиглаж болно.

item8-ыг 1-р мөр, 2-р баганаас эхэлж, 5-р мөр, 6-р баганаар төгсгөнө.
Синтакс: grid-row-start / grid-column-start / grid-row-end / grid-column-end.

.item8 {
grid-area: 1 / 2 / 5 / 6;
}

Naming Grid Items

grid-area шинж чанарыг сүлжээний зүйлсээ (item) нэрлэхэд ашиглаж болно.
Нэрлэсэн сүлжээний зүйлүүдийг сүлжээний контейнерийн grid-template-areas шинж чанарыг ашиглан зааж болно.

item1 нь myArea нэртэй, 5-н баганатай сүлжээний бүх таван багананд байрлана:

.item1 {
grid-area: myArea;
}
.grid-container {
grid-template-areas: 'myArea myArea myArea myArea myArea';
}

.item1 {
grid-area: myArea;
}
.grid-container {
grid-template-areas: 'myArea myArea . . . ';

Бүх item-ийг нэрлэн ашиглахад бэлэн вэб хуудасны загвар үүсгэе:

.item1 { grid-area: header; }
.item2 { grid-area: menu; }
.item3 { grid-area: main; }
.item4 { grid-area: right; }
.item5 { grid-area: footer; }

.grid-container {
grid-template-areas:
'header header header header header header'
'menu main main main right right'
'menu footer footer footer footer footer';
}

Энэхүү сүлжээний бүтэц нь 6-н багана, 3-н мөр агуулсан.

Header
Menu
Main
Right
Footer

The Order of the Items

Grid Layout нь item-үүдийг хаана ч байрлуулах боломжийг олгодог.

HTML код дээр эхний item-ийг сүлжээний эхний item болгон харуулах албагүй.

.item1 { grid-area: 1 / 3 / 2 / 4; }
.item2 { grid-area: 2 / 3 / 3 / 4; }
.item3 { grid-area: 1 / 1 / 2 / 2; }
.item4 { grid-area: 1 / 2 / 2 / 3; }
.item5 { grid-area: 2 / 1 / 3 / 2; }
.item6 { grid-area: 2 / 2 / 3 / 3; }

media queries ашиглан дэлгэцэнд тодорхой хэмжээг заан item-н дарааллыг өөрчилж болно (Цонхны хэмжээг 500 пиксел болгож өөрчилснөөр нөлөөг үзэх):

@media only screen and (max-width: 500px) {
.item1 { grid-area: 1 / span 3 / 2 / 4; }
.item2 { grid-area: 3 / 3 / 4 / 4; }
.item3 { grid-area: 2 / 1 / 3 / 2; }
.item4 { grid-area: 2 / 2 / span 2 / 3; }
.item5 { grid-area: 3 / 1 / 4 / 2; }
.item6 { grid-area: 2 / 3 / 3 / 4; }
}
