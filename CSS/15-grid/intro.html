CSS Grid Layout Module

Grid Layout

CSS Grid Layout модуль нь мөр, багана бүхий сүлжээнд суурилсан байршлын системийг санал болгодог,
энэ нь float болон байрлалыг ашиглахгүйгээр вэб хуудсыг хөгжүүлэхэд хялбар болгодог.

Grid зохион байгуулалт нь нэг буюу хэд хэдэн child элемент бүхий parent элементээс бүрдэнэ.
display шинж чанарыг grid эсвэл inline-grid болгон тохируулсан parent элементтэй байх ёстой.
Grid контейнерийн шууд child элементүүд автоматаар grid item болно.

Энэ grid зохион байгуулалт нь зургаан багана, гурван мөр агуулсан:

Header
Menu
Main
Right
Footer

Display Property

HTML элемент нь display шинж чанарыг grid эсвэл inline-grid гэж тохируулсан тохиолдолд сүлжээний контейнер болдог.

display: grid; ашиглан block-level grid container болсон.

.grid-container {
display: grid;
}

Grid Columns

grid item-н босоо шугмуудыг columns (багана) гэж нэрлэдэг.

Grid Rows

grid item-н хэвтээ шугмуудыг rows (эгнээ) гэж нэрлэдэг.

Grid Gaps

Багана/мөр бүрийн хоорондох зайг gap (хоосон зай) гэж нэрлэдэг.
Дараах шинж чанаруудын аль нэгийг ашиглан зайны хэмжээг тохируулж болно:

  • column-gap
  • row-gap
  • gap

column-gap шинж чанар нь баганын хоорондох зайг тогтоодог.

.grid-container {
display: grid;
column-gap: 50px;
}

Grid Lines

Баганын хоорондох мөрүүдийг column line гэж нэрлэдэг.
Мөр хоорондын шугамыг row line гэж нэрлэдэг.

grid item-ийг сүлжээний контейнерт байрлуулахдаа line-ны дугаарыг бичнэ.

grid item-г 1-р column line-д байрлуулж, 3-р column line-д төгсгөнө:

.item1 {
grid-column-start: 1;
grid-column-end: 3;
}

All CSS Grid Properties

Property Description
column-gap Баганын хоорондох зайг заана.
gap row-gap, column-gap шинж чанаруудын товчилсон шинж чанар.
grid grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns, grid-auto-flow шинж чанаруудын товчилсон шинж чанар.
grid-area grid item нэрийг заах эсвэл энэ шинж чанар нь grid-row-start, grid-column-start, grid-row-end, grid-column-end шинж чанаруудын товчилсон шинж чанар юм.
grid-auto-coliumns Баганын хэмжээг тохируулна. (өгөгдмөл)
grid-auto-flow Автоматаар байрлуулсан item сүлжээнд хэрхэн оруулахыг заана.
grid-auto-rows Мөрийн хэмжээг тохируулна (өгөгдмөл).
grid-column grid-column-start, the grid-column-end шинж чанаруудын товчилсон шинж чанар.
grid-column-end grid item хаана дуусахыг заана.
grid column-gap Багана хоорондох зайны хэмжээг тохируулна.
grid-column-start grid item хаанаас эхлүүлэхийг заана.
grid-gap grid-row-gap, grid-column-gap шинж чанаруудын товчилсон шинж чанар.
grid-row grid-row-start, grid-row-end шинж чанаруудын товчилсон шинж чанар.
grid-row-end grid item хаана дуусахыг зааж өгнө.
grid-row-gap Мөр хоорондын зайны хэмжээг заана.
grid-row-start grid item хаанаас эхлүүлэхийг зааж өгнө
grid-template grid-template-rows, grid-template-columns, grid-areas шинж чанаруудын товчилсон шинж чанар.
grid-template-areas Нэрлэсэн grid item ашиглан багана, мөрийг хэрхэн харуулахыг зааж өгнө.
grid-template-columns Сүлжээний бүтэц дэх баганын хэмжээ болон баганын тоог заана.
grid-template-rows Сүлжээний бүтэц дэх мөрийн хэмжээг зааж өгнө.
row-gap Сүлжээний мөр хоорондын зайг заана.
