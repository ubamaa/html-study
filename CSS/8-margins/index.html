CSS Margins

Margin нь тодорхой хил хязгаараас гадуур байгаа элементүүдийн эргэн тойронд зай үүсгэхэд ашиглагддаг.

CSS Margins (Захын зай)

CSS margin шинж чанарууд нь элементүүдийн эргэн тойронд, тодорхой хилээс гадуур орон зай үүсгэхэд ашиглагддаг.

CSS-ийн тусламжтайгаар захын зайг бүрэн хянах боломжтой.
Элементийн тал (top, right, bottom, left) тус бүрт margin тогтоох шинж чанарууд байдаг.

Margin - Individual Sides

CSS нь элементийн тал бүрийн захын зайг тодорхойлох шинж чанарууд:

  • margin-top (дээд захын зай )
  • margin-right (баруун захын зай)
  • margin-bottom (доод захын зай)
  • margin-left (зүүн захын зай)

Бүх талын шинж чанарууд дараах утгатай байж болно:

  • auto - хөтөч нь захын зайг тооцдог.
  • length - px, pt, cm гэх мэт захын зайг зааж өгнө.
  • % - агуулж буй элементийн өргөний %-ийн ирмэгийг заана.
  • inherit - margin нь эх элементээс өвлөгдөх ёстойг заана.

Margin - Shorthand Property

Кодыг богиносгохын тулд margin шинж чанарыг нэг шинж чанарт зааж өгч болно.

Margin нь дээр дурдсан хувийн (Individual) ​​маржин шинж чанаруудын товчилсон шинж чанар юм.

Тэгэхээр, энэ нь хэрхэн ажилладаг вэ:

Хэрэв margin 4-н утгатай бол:
  • margin: 25px 50px 75px 100px;
    • top margin is 25px
    • right margin is 50px
    • bottom margin is 75px
    • left margin is 100px
Хэрэв margin 3-н утгатай бол:
  • margin: 25px 50px 75px;
    • top margin is 25px
    • right and left margins are 50px
    • bottom margin is 75px
Хэрэв margin 2 утгатай бол:
  • margin: 25px 50px;
    • top and bottom margins are 25px
    • right and left margins are 50px
Хэрэв margin нэг утгатай бол:
  • margin: 25px;
    • all four margins are 25px

The auto Value

Элементийг хайрцагт хэвтээ байдлаар голлуулахын тулд margin шинж чанарыг auto болгоно.
Дараа нь өргөний хэмжээг тохируулж болно, үлдсэн зай нь зүүн болон баруун талаасаа тэнцүү хуваагдана.

<style>
div {
width: 400px;
margin: auto;
border: 1px solid red;
}
</style>

The inherit Value

Энэ жишээнд <p class="ex1"> элементийн margin-left нь эхний (<div>) элементээс залгамжилсан байна.

<style>
div {
border: 1px solid red;
margin-left: 100px;
}

p.ex1 {
margin-left: inherit;
}
</style>

<div>
<p class="ex1">This paragraph has an inherited left margin (from the div element).</p>
</div>

CSS Margin Collapse

Зарим тохиолдолд хоёр захын зай (margin) давхцдаг.

Margin Collapse

Элементүүдийн дээд, доод ирмэг нь заримдаа том хэмжээтэй бол нэг талбайд давхцдаг.
Энэ нь зүүн, баруун талын захад тохиолддоггүй, зөвхөн дээд болон доод захын зайд гардаг!

<style>
h1 {
margin: 0 0 50px 0;
}

h2 {
margin: 20px 0 0 0;
}
</style>

Дээрх жишээнд <h1> элементийн доод захын зай нь 50px, <h2> элементийн дээд ирмэг нь 20px байна.
<h1>, <h2>-ийн хоорондох босоо зай нийт 70px (50px + 20px) байх ёстой!
Иймээс захын зай давхцсаны улмаас бодит margin 50px болно.

All CSS Margin Properties
Property Description
margin Нэг элементэд бүх шинж чанарыг тохируулсан shorthand шинж чанар.
margin-bottom Элементийн доод ирмэгийг тохируулна.
margin-left Элементийн зүүн захыг тохируулна.
margin-right Элементийн баруун захыг тохируулна.
margin-top Элементийн дээд ирмэгийг тохируулна.
