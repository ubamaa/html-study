CSS Borders

border-color хүрээний өнгө тохируулахад ашигладаг.

CSS Border Color

Хүрээний өнгийг дараах байдлаар тохируулж болно:

  • namn - өнгөний нэрийг зааж өгч болно.
  • HEX - HEX утгыг зааж өгөх "#ff0000" г.м.
  • RGB - RGB утгыг зааж өгөх "rgb(255,0,0)" г.м.
  • HSL - HSL утгыг зааж өгөх "hsl(0, 100%, 50%)" г.м.

Note: "border-color" нь ганцаараа ашиглагдахгүй, "border-style" эхэнд тодорхойлогдоно.

Хүрээ - solid, өнгө - улаан.

Хүрээ - solid, өнгө - ногоон.

Хүрээ - dotted, өнгө - хөх.

Specific Side Colors

border-color нь 4-н утгатай (top border, right border, bottom border, and the left border).

Тиймээс 4-н талыг өөр өөр өнгөөр тодорхойлж болно.

Хүрээ - solid, өнгө - multicolor.

HEX Values

Хүрээний өнгийг арван зургаатын тоо (HEX) ашиглан тодорхойлж болно.

Хүрээ solid, өнгө - #ff0000 (red).

RGB Values

Хүрээний өнгийг RGB утгын тусламжтай зааж өгч болно.

Хүрээ - solid, өнгө - blue: rgb(0, 0, 255).

HSL Values

Мөн HSL утгыг ашиглаж болно.

Хүрээ - solid, өнгө - grey: hsl(0, 0%, 73%).
