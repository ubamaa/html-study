CSS Selectors

CSS сонгогч нь таны загварчлахыг хүсэж буй HTML элемент(үүд)-ийг сонгоно.

CSS Selectors

CSS сонгогчдыг таван төрөлд хувааж болно.

  • Simple selectors (name, id, class дээр үндэслэн элементүүдийг сонгох)
  • Combinator selectors (тэдгээрийн хоорондын тодорхой харилцаанд тулгуурлан элементүүдийг сонгох)
  • Pseudo-class selectors (тодорхой төлөвт тулгуурлан элементүүдийг сонгох)
  • Pseudo-elements selectors (элементийн хэсгийг сонгож, загварчлах)
  • Attribute selectors (атрибут эсвэл атрибутын утга дээр үндэслэн элементүүдийг сонгох)

Энэ хуудсанд энгийн CSS сонгогчийг (Simple selectors) тайлбарлах болно.

The CSS element Selector

Элемент сонгогч нь элементийн name дээр үндэслэн HTML элементүүдийг сонгоно.

<style>
p {
text-align: center;
color: red;
}
</style>

The CSS id Selector

id сонгогч нь тодорхой элементийг сонгохдоо HTML элементийн id шинж чанарыг ашигладаг.

id элемент нь өвөрмөц тул id сонгогчийг нэг элементэд сонгож ашигладаг!

Тодорхой id-тай элементийг сонгохын тулд хэш (#) тэмдэг бичиж, элементийн id-г оруулна.

Дараах CSS дүрмийг id="para1" бүхий HTML элементэд хэрэглэнэ:

<style>
#para1 {
text-align: center;
color: red;
}
</style>
<p id="para1">Hello World!</p>

The CSS class Selector

class сонгогч нь тодорхой class шинж чанартай HTML элементүүдийг сонгоно.

Тодорхой class-тай элементүүдийг сонгохын тулд цэг (.) тэмдэг, дараа нь class нэрээ бичнэ.

Жишээ 1.

Энд class="center"-тэй HTML элементүүд улаан өнгөтэй, төвд зэрэгцсэн байна:

<style>
.center {
text-align: center;
color: red;
}
</style>
<h1 class="center">Red and center-aligned heading</h1>
<p class="center">Red and center-aligned paragraph.</p>

Жишээ 2.

Мөн class нь зөвхөн тодорхой HTML элементүүдэд нөлөөлөх ёстойг зааж болно.

Энэ жишээнд class="center"-тэй зөвхөн <p> элементүүд улаан өнгөтэй, төвд зэрэгцсэн байна:

<style>
p.center {
text-align: center;
color: red;
}
</style>
<h1 class="center">This heading will not be affected</h1>
<p class="center">This paragraph will be red and center-aligned.</p>

Жишээ 3.

HTML элементүүд нь нэгээс олон class-д нөлөөлөх боломжтой.

Энэ жишээнд <p> элементийг class="center", class="large" загварын дагуу загварчилсан байна:

<style>
p.center {
text-align: center;
color: red;
}

p.large {
font-size: 300%;
}
</style>

<h1 class="center">This heading will not be affected</h1>
<p class="center">This paragraph will be red and center-aligned.</p>
<p class="center large">This paragraph will be red, center-aligned, and in a large font-size.</p>

The CSS Universal Selector

Universal Selector нь хуудасны бүх HTML элементүүдийг сонгоно.
Үүний тулд (*) тэмдэг ашиглана.

<style>
* {
text-align: center;
color: teal;
}
</style>

Universal Selector-ээр энэ хуудасны загварыг тодорхойлов!

The CSS Grouping Selector

Grouping Selector нь ижил загварын тодорхойлолт бүхий HTML элементүүдийг сонгоно.
Бүлэглэхийн тулд элемент бүрийг таслалаар (,) тусгаарлана.

Дараах CSS кодыг харцгаая (h1, h2, p ижил загварын тодорхойлолттой):

<style>
h1, h2, p {
text-align: center;
color: red;
}
</style>

Бүлэглэх нь бичиглэлийг багасгахад тустай.
