CSS Selectors

CSS сонгогчдыг таван төрөлд хувааж болно.

Simple selectors - Энгийн сонгогчид (name, id, class элементүүд дээр үндэслэн сонгох)

Энэ хуудсанд хамгийн энгийн CSS сонгогчийг (Simple selectors) тайлбарлах болно.

The CSS id Selector

Ногоон өнгөтэй 3-н мөрөнд id selector ашигласан.

id элементийг сонгохын тулд хэш (#) тэмдэгт бичиж, элементийн id-г оруулна.

The CSS class Selector

Энэхүү 3-н мөрөнд class selector ашигласан.

Class элементийг сонгохын тулд (.) тэмдэгт бичиж, дараа нь class нэрийг бичнэ.

Class selector нь хэмжээ, байрлал мөн элементийг сонгон зааж өгч болно.

Энд p элементийг class="left" болон class="large"-ийн дагуу загварчилсан байна.
