CSS External

link элемент доторх External style файлыг .css өргөтгөлөөр хадгалах ёстой.

CSS Internal

Internal style-ийг head хэсгийн style элемент дотор тодорхойлно.

Inline CSS

Нэг элементэд өвөрмөц style хэрэглэхийн тулд Inline style ашиглаж болно.

Multiple Style Sheets (Олон загварын хуудас)

Хэрэв нэг хуудсанд ижил selector (элемент)-үүдийг сонгосон бол хамгийн сүүлд бичсэн style-ийн утга үйлчилнэ.

Жишээ: Энд эхний бичсэн style External style, дараа нь Internal style бичихэд сүүлчийнхээр нь тодорхойлогдсон байна.

Cascading Order (Олон style-ийг нэг дор нэгтгэсэн)

HTML элементэд нэгээс олон загвар заасан тохиолдолд ямар хэв маягийг ашиглах вэ?

Бүх style нэг хуудсанд «каскад» болоход "эхнийх нь хамгийн давуу эрхтэй" гэсэн дүрмийн дагуу байх болно.

  1. Inline style (HTML элемент дотор)
  2. External and Internal style sheets (in the head section)
  3. Browser default

Тиймээс inline style нь хамгийн чухал ач холбогдолтой, External болон Internal styles, хөтчийн өгөгдмөл тохиргоог хүчингүй болгодог.
