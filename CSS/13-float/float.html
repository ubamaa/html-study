CSS Layout - float and clear

CSS float шинж чанар нь элемент хэрхэн хөвөхийг зааж өгдөг.

CSS clear шинж чанар нь clear элементийн дэргэд, аль талд нь ямар элементүүд байрлаж болохыг заадаг.

The float Property

float шинж чанар нь агуулгыг байршуулахад, форматлахад ашигладаг.
(Жишээ нь: дүрсийг блокон текстийн зүүн талд байршуулж болох эсэх).

Float шинж чанар нь дараах утгуудыг авч болно:

  • left - элемент нь агуулгын зүүн талд шилжсэн.
  • right - элемент нь агуулгын баруун талд шилжсэн.
  • none - Энэ нь анхдагч өгөгдөл, байрлал заахгүй (зөвхөн текст байгаа газарт байрлана).
  • inherit - толгой элементийн утгыг өвлөх болно./li>

Энгийнээр хэлбэл, float шинж чанарыг текстийн эргэн тойронд зургаар хүрээлүүлэхэд ашиглаж болно.

Example - float: right;

Дараах жишээ нь зураг текстийн баруун талд байрласныг харуулж байна.

<style>
img {
float: right;
}
</style>

<p><img src="pineapple.jpg" alt="Pineapple" style="width:170px; height:170px; margin-left:15px;"> ... </p>


Example - float: left;

Дараах жишээн дээр зураг текстийн зүүн талд байрласныг харуулж байна.

<style>
img {
float: left;
}
</style>

<p><img src="pineapple.jpg" alt="Pineapple" style="width:170px; height:170px; margin-right:15px;"> ... </p>



Example - No float

Дараах жишээн дээр зураг нь текстийн яг байгаа газарт гарч ирж байгаа (float: none;):

<style>
img {
float: none;
}
</style>

<p><img src="pineapple.jpg" alt="Pineapple" style="width:170px; height:170px;"> ... </p>

Example - Float Next To Each Other

div элементүүд ихэвчлэн бие биенийхээ дээр байрладаг. Хэрэв float: left-ийг ашиглавал элементүүдийг зэрэгцүүлэн байрлуулж болно.

All CSS Float Properties

Property Description
box-sizing Элементийн өргөн, өндрийг хэрхэн тооцоолох, тэдгээрт дэвсгэр болон хүрээ оруулах эсэхийг зааж өгнө.
clear float элементтэй зэрэгцсэн элементийг хэрхэн байрлахыг зааж өгнө.
float Элемент зүүн, баруун талд байрлах эсвэл огт хөдлөхгүй байхыг зааж өгнө.
overflow Агуулга нь элементийн ирмэгээс халихад хэрхэн шийдэхийг заах болно.
overflow-x Хэрэв агуулгын зүүн/баруун ирмэг элементийн агуулгын талбараас гарвал хэрхэн засахыг заана.
overflow-y Агуулгын дээд/доод ирмэг нь элементийн агуулгын талбараас давах үед хэрхэн засахыг заана.
