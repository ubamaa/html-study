CSS Layout - float and clear

CSS float шинж чанар нь элемент хэрхэн хөвөхийг зааж өгдөг.

CSS clear шинж чанар нь цэвэрлэсэн элементийн дэргэд, аль талд нь ямар элементүүд байрлаж болохыг заадаг.

The float Property

float шинж чанарыг агуулгыг байршуулахад, форматлахад ашигладаг.
(Жишээ нь: дүрсийг блокон текстийн зүүн талд байршуулж болох эсэх).

Float шинж чанар нь дараах утгуудаас авч болно:

  • left - элемент нь агуулгын зүүн талд шилжсэн.
  • right - элемент нь агуулгын баруун талд шилжсэн.
  • none - Энэ нь анхдагч өгөгдөл, байрлал заахгүй (зөвхөн текст байгаа газарт байрлана).
  • inherit - толгой элементийн утгыг өвлөх болно./li>

Энгийнээр хэлбэл, float шинж чанарыг текстийн эргэн тойронд зургаар хүрээлүүлж болно.

Example - float: right;

Дараах жишээ нь зураг текстийн баруун талд байрласныг харуулж байна.

<style>
img {
float: right;
}
</style>

<p><img src="pineapple.jpg" alt="Pineapple" style="width:170px; height:170px; margin-left:15px;"> ... </p>

Example - float: left;

Дараах жишээн дээр зураг текстийн зүүн талд байрласныг харуулж байна.

<style>
img {
float: left;
}
</style>

<p><img src="pineapple.jpg" alt="Pineapple" style="width:170px; height:170px; margin-right:15px;"> ... </p>

Example - No float

Дараах жишээн дээр зураг нь текстийн яг байгаа газарт гарч ирж байгаа (float: none;):

<style>
img {
float: none;
}
</style>

<p><img src="pineapple.jpg" alt="Pineapple" style="width:170px; height:170px;"> ... </p>

Example - Float Next To Each Other
