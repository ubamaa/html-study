CSS Layout - clear and clearfix

The clear Property

float шинж чанарыг ашиглаж байхад дараагийн элементийн байрлалыг (баруун, зүүн талд биш) өөрчлөхийг хүсвэл clear шинж чанарыг ашиглах хэрэгтэй болно.

clear шинж чанар нь float элементийн хажууд байгаа элементийг хэрхэн байршуулахыг зааж өгдөг.

clear шинж чанар нь дараах утгуудын аль нэгийг агуулж болно:

  • none - Элементийг зүүн, баруун float элементийн доор байрлуулаагүй. Энэ нь анхдагчаар байгаа.
  • left - Элементийг float: left; элементийн доор байрлуулна.
  • right - Элементийг float: right; элементийн доор байрлуулна.
  • both - Элементийг зүүн болон баруун float элементийн доор байрлуулна.
  • inherit - Элемент нь өмнөх элементээсээ clear утгаа авдаг.

float засахдаа clear-ийг float-тэй тааруулах хэрэгтэй:
Хэрэв элемент float: left; байвал clear: left; ашиглах ёстой,
float элемент байрлалаа хадгалж, харин clear элемент хуудсан дээр түүний доор гарч ирнэ.

div1 (Without clear)
div2 - HTML-д div1-ийн дараа div2 байдгийг анхаарах! div1 нь float: left; утгатай тул div2 текстэд div1 багтсан байна.

div3 (With clear)
div4 - энд clear: left; утгатай учир div3-н доор байрласан. clear: left; утга нь зүүн байрлалыг зааж байна. Мөн clear: right болон both утгуудыг ашиглаж болно.
