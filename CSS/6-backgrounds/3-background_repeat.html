CSS Background Image Repeat

CSS background-repeat

background-repeat: нь дэвсгэрийн давталтыг харуулна.

background-repeat: repeat-x; - Зургийг хэвтээ байдлаар давтахын тулд тохируулна.

background-repeat: repeat-y; - Зургийг босоо байдлаар давтахын тулд тохируулна.

background-repeat: no-repeat; - Дэвсгэр зургийг зөвхөн нэг удаа давтах шинж чанараар тодорхойлогдсон.

CSS background-position

background-position: - Энэ шинж чанарыг дэвсгэр зургийн байрлалыг тодорхойлоход ашигладаг.

CSS Background Attachment

CSS background-attachment

background-attachment: - Энэ шинж чанар нь дэвсгэр зургийг гүйлгэх эсвэл хуудасны үлдсэн хэсэгтэй хамт гүйлгэхгүй байх эсэхийг тодорхойлдог.

background-attachment: scroll; - дэвсгэр зургийг гүйлгэнэ.

background-attachment: fixed; - дэвсгэр зураг гүйхгүй.
