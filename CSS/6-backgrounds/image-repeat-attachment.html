CSS Background Image

CSS background-image

Background-image шинж чанар нь элементийн дэвсгэр болгон ашиглах зургийг зааж өгдөг.
Анхдагч байдлаар, дэвсгэр зураг давтагддаг тул бүтэн элементийг хамарна.

Арын дэвсгэр зургийг <p> элемент гэх мэт тодорхой элементэд тохируулж болно:

Текст болон дэвсгэр зургийн хослолыг зөв тохируулахгүй бол текстийг уншихад хэцүү болно.
Note: Дэвсгэр зургийг сонгохдоо текстэнд саад учруулахгүй зургийг ашиглах хэрэгтэй.


The CSS Background Image Property

Property Description
background-image Элементийн дэвсгэр зургийг тохируулна.


CSS Background Image Repeat

CSS background-repeat

Анхдагч байдлаар, background-image шинж чанар нь дэвсгэр зургийг хуудасны зүүн дээд буланд байрлуулж, босоо болон хэвтээ байдлаар давтдаг.

background-repeat: нь дэвсгэрийн давталтыг тохируулна:

background-repeat: no-repeat; - Дэвсгэр зургийг зөвхөн нэг удаа давтах шинж чанараар тодорхойлогдсон.

background-repeat: repeat-x; - Зураг хэвтээ байдлаар давтагдана.

background-repeat: repeat-y; - Зураг босоо байдлаар давтагдана.

Зарим зургийг зөвхөн хэвтээ эсвэл босоо байдлаар давтахгүй бол дэвсгэр хачирхалтай харагддаг. Энэ нь текст болон зургийн хослолыг илүү сайн харагдуулахад тустай.

CSS background-position

background-position: шинж чанарыг дэвсгэр зургийн байрлалыг тодорхойлоход ашигладаг.

Энэ жишээн дээр дэвсгэр зурагт зөвхөн 1 удаа давтагдаж, баруун талд байрлан, текстэд баруун захын зай нэмсэн байна.

<style>
body {
background-image: url("img_tree.png");
background-repeat: no-repeat;
background-position: right top;
margin-right: 200px;
}
</style>

The CSS Background Repeat and Position Properties

Property Description
background-position Арын дэвсгэр зургийн эхлэлийн байрлалыг тохируулна.
background-repeat Арын дэвсгэр зураг хэрхэн давтагдахыг тохируулна.


CSS Background Attachment

CSS background-attachment

background-attachment: шинж чанар нь дэвсгэр зургийг гүйлгэх эсвэл байрлалаа хадгалах эсэхийг заадаг
(хуудасны үлдсэн хэсэгтэй хамт гүйлгэхгүй).

background-attachment: fixed; - дэвсгэр зургийг хуудастай хамт гүйлгэнэ.

<style>
body {
background-image: url("img_tree.png");
background-repeat: no-repeat;
background-position: right top;
margin-right: 200px;
background-attachment: fixed;
}
</style>

background-attachment: scroll; - дэвсгэр зураг хуудастай гүйхгүй.

<style>
body {
background-image: url("img_tree.png");
background-repeat: no-repeat;
background-position: right top;
margin-right: 200px;
background-attachment: scroll;
}
</style>

The CSS Background Attachment Property

Property Description
background-attachment Дэвсгэр зургийг хуудастай хамт гүйлгэх эсэхийг тохируулна.
