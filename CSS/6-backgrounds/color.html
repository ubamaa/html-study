CSS Backgrounds

CSS background шинж чанар нь элементүүдэд дэвсгэр эффект нэмэхэд ашиглагддаг.

CSS background дараах шинж чанаруудтай:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background (shorthand property)

CSS background-color

background-color шинж чанар нь элементийн дэвсгэр өнгийг тодорхойлдог.
CSS тусламжтайгаар өнгийг дараах өнгөнүүдээс сонгож болно:

- Өнгөний нэрээр (улаан, шар гэх мэт).
- HEX утга ("#ff0000")
- RGB, RGBA утга "rgb (255,0,0)"

Other Elements

Ямар ч HTML элементийн дэвсгэр өнгийг тохируулж болно:
Энэ хуудасны <h1> <p> <div> элементүүд өөр өөр дэвсгэр өнгөтэй байна.

Opacity/Transparency

Opacity шинж чанар нь элементийн харанхуй/тод байдлыг тодорхойлдог.
0.0-ээс 1.0 хүртэлх утгыг авч болно. Утга бага байх тусам цайвар болно.

opacity 0.1 (Opacity-н үр дүн)

opacity 0.3 (Opacity-н үр дүн)

opacity 0.6 (Opacity-н үр дүн)

opacity 1 (default) (Opacity-н үр дүн)

Note: Элементийн дэвсгэр дээр тунгалаг байдал нэмж, opacity шинж чанарыг ашиглах үед түүний child элементүүд мөн тунгалаг байдлыг өвлөнө.
Энэ нь тод элемент доторх текстийг уншихад хүндрэл учруулдаг.

Transparency using RGBA

Хэрэв дээрх жишээний child элементүүдэд opacity ашиглахыг хүсэхгүй бол RGBA өнгөний утгыг ашиглаж болно.
Доорх жишээ нь opacity-ийг текстийн бус дэвсгэр өнгөнд тохируулсан:
Opacity шинж чанарыг ашиглах үед текст ил тод болж, дэвсгэрийн өнгө хэрхэн харагдахыг анхаарах!

RGBA()-ын үр дүн:

10% opacity

30% opacity

60% opacity

default

RGB-ээс гадна альфа суваг (RGB A) бүхий RGB өнгөний утгыг ашиглаж болно.
Альфа параметр нь 0.0 (бүрэн цайвар) - 1.0 (тунгалаг биш) хоорондох тоо юм.
