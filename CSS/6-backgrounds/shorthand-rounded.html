CSS Background Shorthand

CSS background - Shorthand property

Кодыг богиносгохын тулд бүх дэвсгэр шинж чанарыг нэг шинж чанарт зааж өгөх боломжтой.
Үүнийг Shorthand property гэж нэрлэдэг.

body {
background-color: #ffffff;
background-image: url("img_tree.png");
background-repeat: no-repeat;
background-position: right top;
}
гэж бичихийн оронд background товчилсон шинж чанарыг ашиглан:

body {
background: #ffffff url("img_tree.png") no-repeat right top;
margin-right: 200px;
}

нэг илэрхийлэлд бүх дэвсгэр шинж чанарыг тохируулж болно.

Shorthand property ашиглахад утгын дараалал дараах байдалтай байна:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

Дээрх шинж чанаруудын аль нэг нь байхгүй байж болно, бусад нь энэ дарааллаар байх ёстой.
Гэвч энд background-attachment шинж чанар ашиглагддаггүй гэдгийг анхаарах!


All CSS Background Properties

Property Description
background Бүх дэвсгэр шинж чанарыг нэг илэрхийлэлд тохируулна.
background-attachment Дэвсгэр зургийг хуудасны бусад хэсэгтэй хамт гүйлгэх эсэхийг тохируулна.
background-clip Арын дэвсгэр зургийн талбайг зааж өгнө.
background-color Элементийн дэвсгэр өнгийг тохируулна.
background-image Элементийн дэвсгэр зургийг тохируулна.
background-origin Дэвсгэр зургийн гарал үүслийн байрлалыг (арын байрлал тогтоох хэсэг) зааж өгдөг.
background-position Арын дэвсгэр зургийн эхлэлийн байрлалыг тохируулна.
background-repeat Арын дэвсгэр зураг хэрхэн давтагдахыг тохируулна.
background-size Арын дэвсгэр зургийн хэмжээг зааж өгнө.
