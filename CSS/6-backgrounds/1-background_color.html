CSS Backgrounds

CSS background шинж чанар нь элементүүдэд дэвсгэр эффект нэмэхэд ашиглагддаг.

CSS background дараах шинж чанаруудтай:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background(shorthand property)

CSS background-color

background-color шинж чанар нь элементийн дэвсгэр өнгийг тодорхойлдог.

CSS тусламжтайгаар өнгийг дараах өнгөнүүдээс сонгож болно:

- Өнгөний нэрээр (улаан, шар гэх мэт).
- HEX утга ("#ff0000")
- RGB, RGBA утга "rgb (255,0,0)"

Other Elements

Ямар ч HTML элементийн дэвсгэр өнгийг тохируулж болно:
Энэ хуудасны <h1>, <p>, <div> элементүүд өөр өөр дэвсгэр өнгөтэй байна.
