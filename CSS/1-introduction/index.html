CSS Introduction

CSS бол вэб хуудасны загвар гаргахад ашигладаг хэл юм.

What is CSS?

  • CSS нь Cascading Style Sheets үгийн товчлол юм.
  • CSS нь HTML элементүүдийг дэлгэц, бусад медиа дээр хэрхэн харуулахыг тайлбарладаг.
  • CSS нь маш их ажлыг хялбарчилдаг. Энэ нь олон вэб хуудасны байршлыг нэгэн зэрэг хянах боломжтой.
  • External загварын хүснэгтүүд нь CSS файлд хадгалагддаг.

Why Use CSS?

CSS нь вэб хуудсуудыг загварчлахад ашигладаг:
үүнд дизайн, байршил, янз бүрийн төхөөрөмж, дэлгэцийн хэмжээ сонголтууд орно.

CSS Solved a Big Problem

HTML нь хэзээ ч вэб хуудсыг форматлах tag агуулж байгаагүй!
HTML нь вэб хуудасны агуулгыг тайлбарлах зорилгоор бүтээгдсэн.

Хуудас бүрт үсгийн фонт, өнгөний мэдээллийг нэмдэг томоохон вэбсайтуудыг хөгжүүлэх нь хугацаа болон өртөгтэй үйл явц болжээ.
Энэ асуудлыг шийдэхийн тулд World Wide Web Consortium (W3C) CSS-ийг бүтээсэн.

CSS Saves a Lot of Work!

Загварын тодорхойлолтыг ихэвчлэн extarnal .css файлд хадгалдаг.

Extarnal загварын хуудсан дахь файлын тусламжтайгаар бүхэл вэбсайтын харагдах байдлыг өөрчлөх боломжтой!


CSS Syntax

CSS дүрэм (rule) нь сонгогч (selector) болон илэрхийллийн блокоос (declaration block) бүрдэнэ.

h1 {color:gold; font-size:15px;}

Сонгогч нь таны загварчлахыг хүсэж буй HTML элементийг заана.

h1 - selector

Илэрхийллийн блок нь цэг таслалаар тусгаарлагдсан 1 болон хэд хэдэн илэрхийллийг агуулна.

( {color:gold; font-size:15px;} ) - declaration

Олон CSS илэрхийллүүдийг цэг таслалаар тусгаарлаж, илэрхийллийн блокуудыг {} хаалтанд бичнэ.

({color:gold}, {font-size:15px;}) - 2 declaration

Илэрхийлэл бүр нь CSS шинж чанар болон түүний утгыг агуулна.

(color: ; font-size: ) - property

(gold, 15px) - value
