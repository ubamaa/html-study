CSS Layout - Overflow

Overflow CSS шинж чанар нь тухайн талбайд багтахгүй том контентыг хэрхэн багтаах аргыг заадаг.

CSS Overflow

Overflow шинж чанар нь контентийн агуулга заасан хэсэгт багтахааргүй бол контентыг таслах эсвэл гүйлгэх зурвас нэмэх эсэхийг тодорхойлдог.

Overflow шинж чанар нь дараах утгатай байна:

  • visible - Өгөгдмөл. Агуулгын халилтыг таслахгүй. Агуулгыг блокийн захаас гадуур харуулна.
  • hidden - агуулгын халилтыг тайрч, энэ хэсэг харагдахгүй болно.
  • scroll - агуулгын халилт таслагдаж, энэ хэсгийг харуулахын тулд гүйлгэх зурвас нэмнэ.
  • auto - scroll-тэй төстэй боловч зөвхөн шаардлагатай үед гүйлгэх мөрийг нэмнэ.

Note: Overflow шинж чанар нь зөвхөн заасан өндөртэй block түвшний элементүүдэд ажилладаг.

overflow: visible

Өгөгдмөл, overflow: visible; нь агуулгын халилтыг таслахгүй, харин блокийн ирмэгийг давж харагдах болно.

Хэрэв та форматаа илүү хянахыг хүсвэл overflow шинж чанарыг ашиглаж болно. Overflow шинж чанар нь агуулга элементийн блокоос хэтэрсэн тохиолдолд юу болохыг зааж өгдөг.


<style>
div {
background-color: coral;
width: 200px;
height: 65px;
border: 1px solid;
overflow: visible;
}
</style>

overflow-x and overflow-y

overflow-x: overflow-y: шинж чанарууд нь агуулгын халилтыг зөвхөн хэвтээ болон босоо чиглэлд өөрчлөх эсэхийг тодорхойлдог.

overflow-x - агуулгын зүүн/баруун талын халилтыг зааж өгдөг.
overflow-y - агуулгын дээд/доод талын халилтыг зааж өгдөг.

Хэрэв та форматаа илүү хянахыг хүсвэл overflow шинж чанарыг ашиглаж болно. Overflow шинж чанар нь агуулга элементийн блокоос хэтэрсэн тохиолдолд юу болохыг зааж өгдөг.

<style>
background-color: coral;
width: 200px;
height: 65px;
border: 1px solid black;
div {
overflow-x: hidden; /* Hide horizontal scrollbar */
overflow-y: scroll; /* Add vertical scrollbar */
}
</style>

All CSS Overflow Properties
Property Description
overflow Агуулга нь элементийн захаас хэтэрсэн тохиолдолд хэрхэн шийдэхийг зааж өгнө.
overflow-wrap Хөтөч нь урт үгтэй мөрүүдийг агуулж байвал тэдгээрийг таслах эсэхийг заана.
overflow-x Хэрэв элементийн агуулга заасан талбайгаас гарвал зүүн/баруун захад халилт гаргахыг зааж өгнө.
overflow-y Элементийн агуулга талбайн гадуур гарах тохиолдолд агуулгын дээд/доод захад халилт хийхийг зааж өгнө.
